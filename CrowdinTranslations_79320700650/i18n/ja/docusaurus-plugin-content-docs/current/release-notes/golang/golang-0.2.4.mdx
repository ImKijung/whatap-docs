2023年11月22日

- <Status>Changed</Status>テキストキャッシュの保存期間を従来の１日おきから１時間おきに変更

- <Status>Changed</Status>kubernates環境でcpu、memeory指標を収集しないように変更、Kubernates エージェントが収集したシステム指標に置き換え

- <Status>Fixed</Status>CentOS 9以降でwhatap-agentサービス実行エラーの修正

- <Status>Fixed</Status>ドッカーコンテナ環境におけるvCPu情報の重複収集現象の修正

- <Status>Fixed</Status>クライアントIPアドレス収集中にポート情報が一緒に収集されるエラーを修正
