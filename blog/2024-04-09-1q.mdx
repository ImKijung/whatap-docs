## `Log`

<div class='indentTab'>

- <Status>Feature</Status> <Cmdname sid="log_search" className="uitext" />, <Cmdname sid="logTail" className="uitext" /> 메뉴에서 로그 검색 필터 기능 개편

  - 검색값 입력 시 지원하는 검색 문법 안내 가이드 기능 추가
  - 추가된 태그 수정 시 키보드 입력으로 수정할 수 있는 기능 추가
  - 태그를 키보드 입력으로 생성 및 수정 가능하고, 연산자는 클릭하여 수정할 수 있는 기능 추가

- <Status>Feature</Status> <Cmdname sid="event_statistic" className="uitext" /> &gt; <Cmdname sid="event_setting" className="uitext" /> 메뉴에서 실시간 로그 이벤트를 추가 또는 수정하면서 API 요청을 실패할 경우, 실패 원인을 메시지로 노출하도록 개선

- <Status>Feature</Status> 에이전트 옵션이 설정된 경우 로그 레벨을 표시하는 색상 표시 기능 추가

