## `Application`

와탭 2024년 2분기 `Application` 부분의 주목할 만한 신규 기능으로 지난 1분기에 이어 ***트랜잭션 맵***의 고도화와 ***애플리케이션 대시보드*** 기본 프리셋 변경 기능을 소개합니다.

***트랜잭션 맵*** 고도화 작업의 일환으로 응답 시간 분포도 차트에서 선택한 항목만을 조회할 수 있는 커스텀 필터 기능을 새롭게 제공합니다. 기존 ***TOP 30*** 섹션 옆의 ***커스텀*** 탭에서 조회를 원하는 특정 트랜잭션, 도메인, IP 항목에서 원하는 값을 직접 입력하거나 ***TOP 30*** 목록에서 제공하는 항목을 선택하세요. 선택한 항목들의 응답 시간 분포도가 차트에 실시간으로 필터링됩니다. 

***애플리케이션 대시보드*** 메뉴로 처음 진입했을 때 조회할 수 있는 대시보드 기본 프리셋을 변경할 수 있습니다. 위젯 선택과 배치를 원하는대로 수정 후 대시보드 오른쪽 위에 있는 프리셋 선택 상자에서 ***기본으로 표시*** 옵션을 활용해 보세요. 기본으로 지정한 대시보드는 프리셋 목록에서 <code class='default-tag'>기본</code> 태그가 표시됩니다.

## `Database`

와탭 2024년 2분기 `Database` 부분에서 주목할 만한 신규 기능 및 변경 사항으로 **Oracle** 상품 <code class="my">V2</code> 출시와 **Oracle Pro** 베타 출시에 대해 안내합니다. 

와탭 데이베이스 상품군이 <code class="my">V2</code>로 전환되고 있습니다. 1분기 **Redis**와 **MongoDB**에 뒤이어 2분기는 **Oracle** <code class="my">V2</code> 서비스가 시작되었습니다. Oracle <code class="my">V2</code>는 데이터 전송팩이 tag count로 바뀌고 UI가 개선되었습니다. 

3분기 정식 출시를 앞두고 **Oracle Pro** 베타 서비스가 시작되었습니다. **Oracle Pro**는 데이터베이스 서버의 메모리에 직접 접근해 성능 지표를 모니터링하는 기술을 적용하였습니다. 최신 데이터를 반영하여 정확한 성능 정보를 1초에 최대 20번까지 모니터링할 수 있습니다. 디스크 I/O를 거치지 않고 메모리에 직접 접근하여 시스템에 부하가 없습니다. 특정 쿼리의 실행 시간, 버커 캐시 상태 등 정밀하게 수집한 성능 정보를 바탕으로 문제를 빠르고 정확하게 분석할 수 있습니다.

* Oracle Pro

* Oracle <code class="my">V2</code>

* 멀티 인스턴스 모니터링 메뉴 사용자 편의성 개선
