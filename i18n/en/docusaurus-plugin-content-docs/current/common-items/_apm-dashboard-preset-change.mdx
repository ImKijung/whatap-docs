### Changing the default preset {#changepreset}

You can change the preset displayed when you first enter the <Cmdname sid="side_appDashboard" className="uitext" /> menu.

1. Check the preset selection box on the upper right of the dashboard.

2. Hover your mouse over a desired preset to set it as default.

   <ImgLang img="apm-dashboard-preset-change-default.png" desc="Preset" />

3. If the <Cmdname sid="BTN08106" className="uitext" /> button appears, select it.

The selected preset is changed to the default preset, and the <Cmdname sid="DSC08162" className="uitext" /> tag appears in the preset list.

<ImgLang img="apm-dashboard-preset-changed-default.png" desc="Preset" />
