## Viewing the past data

<Cmdname sid="application_dashboard" className="uitext" /> basically provides real-time monitoring features, in which you can view the past data.

|                    Real-time mode                    |                       Non-real-time mode                      |
| :--------------------------------------------------: | :-----------------------------------------------------------: |
| ![Real-time mode](/img/apm-timeslector-realtime.png) | ![Non-real-time mode](/img/apm-timeslector-none-realtime.png) |

1. Select ![Pause icon](/img/ico-pause.svg) in the time selector at the upper left of the screen.

2. Click the date and time to set the desired time zone.

   <ImgLang img='apm-timeselector-setting.png' desc='Configure manual time' />

3. Select <Cmdname sid="apply" className="uitext" />.

It updates the data in widgets placed on the dashboard based on the set time. To switch to the Realtime mode, select ![Playback icon](/img/ico-play-time.svg).
