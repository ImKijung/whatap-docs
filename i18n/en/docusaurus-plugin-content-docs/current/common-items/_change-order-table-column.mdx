1. Select ![Configuration icon](/img/ico-setting.svg) on the upper right of the table.

2. In the drop-down menu, select <Cmdname sid="column_settings" className="uitext" />.

3. If the <Cmdname sid="column_settings" className="uitext" /> window appears, drag a desired item in the <Cmdname sid="column_order_setting" className="uitext" /> section for reordering.

4. After all settings are finished, select ![Close icon](/img/ico-close.svg).

The set content is applied to the table.
