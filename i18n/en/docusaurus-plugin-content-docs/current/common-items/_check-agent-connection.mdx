## Checking the agent connection

Go to [Whatap Monitoring Service](https://service.whatap.io). After logging in, select a created project from the project list. If the data of the server connected by the <Cmdname sid="side_appDashboard" className="uitext" /> menu appears through the widget, all settings are completed.
