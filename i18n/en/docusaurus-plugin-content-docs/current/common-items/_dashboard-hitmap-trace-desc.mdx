Through the **_Hitmap_** widget in **_Application dashboard_**, you can open the **_Trace analysis_** window to check the transaction list and step information of each transaction at once. Drag the desired area on the hitmap widget. The **_Trace analysis_** window appears that contains the transaction list and sub-steps of the selected area.

:::tip

To classify and see only the transactions with errors, select **_Error_** on the upper right of the **_Hitmap_** widget in **_Application Dashboard_** and then drag the chart area.

:::
