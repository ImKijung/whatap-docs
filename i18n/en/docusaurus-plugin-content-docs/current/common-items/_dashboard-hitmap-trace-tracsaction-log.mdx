### Transaction Log

To see the logs of the transaction, select the **_Transaction log_** tab.

![Transaction log](/img/analysis-tx-transaction-log.png)

-   **Keyword search**: Enter the desired text to search in the keyword search field, and then press Enter or select the ![Search icon](/img/ico-search.svg) button. It highlights the keywords that match the entered text.

-   ![Highlight icon](/img/ic-highlight.svg) **_Keyword highlight_**: If you set the keyword and color, the texts that match the keyword are automatically highlighted in the log list.

-   ![Configuration icon](/img/ico-setting.svg) **_Table settings_**: You can set the header columns in the log list table.

:::note

-   To enable the **_Transaction log_** tab, apply the agent setting options related to the logs. For more information about the log settings, see [the following](../log/log-function).

-   The **_Transaction log_** tab can be accessed by only the members with the **View log** role. For more information about the member roles, see [the following](../project/project-structure#member-auth).

:::
