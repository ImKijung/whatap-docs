### Transaction log {#anlaysis-trace-transaction-log}

To see the logs of the transaction, select the <Cmdname sid="TTL06128" className="uitext" /> tab.

<ImgLang img="analysis-tx-transaction-log.png" desc="Transaction log" />

- **Keyword search**: Enter the desired text to search in the keyword search field, and then press Enter or select the ![Search icon](/img/ico-search.svg) button. It highlights the keywords that match the entered text.

- ![Highlight icon](/img/ic-highlight.svg) <Cmdname sid="keyword_highlight" className="uitext" />: If you set the keyword and color, the texts that match the keyword are automatically highlighted in the log list.

- ![Configuration icon](/img/ico-setting.svg) <Cmdname sid="table_settings" className="uitext" />: You can set the header columns in the log list table.

:::note

- To enable the <Cmdname sid="TTL06128" className="uitext" /> tab, apply the agent setting options related to the logs. For more information about the log settings, see [the following](../log/log-function).

- The <Cmdname sid="TTL06128" className="uitext" /> tab can be accessed by only the members with the <Cmdname sid="log_read" className="b600" /> role. For more information about the member roles, see [the following](../project/project-structure#member-auth).

:::
