## Viewing the past data

<Cmdname sid="transaction_map" className="uitext" /> provides the real-time monitoring by default. By default, the data can be viewed in real time for 5 minutes. Click the green button in the time selector to select a desired lookup time.

To view the historical data, select ![Pause icon](/img/ico-pause.svg) in the time selector. It changes to non-real-time mode and you can view the transaction data for up to 10 minutes.

<ImgLang img='txmap-past-time-selector.png' desc='과거 시간 조회' />

Set the desired date and time and then select <Cmdname sid="apply" className="uitext" />. Based on the time set by the user, the chart data of <Cmdname sid="transaction_map" className="uitext" /> is updated.

:::note

- The range of the historical inquiries may differ depending on the transaction volume.

- To switch to the Realtime mode after a historical inquiry, select ![Playback icon](/img/ico-play-time.svg).

:::
