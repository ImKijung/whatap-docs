### Checking the SQL statistics

You can see the list of top SQLs for the time period along with the trend of key performance metrics that occurred during the specific time in the past.

<ImgLang img='db-instancemonitoring2sqlstatics.png' desc='SQL 통계 바로가기' />

1. In <Cmdname sid="time" className="uitext" />, select <LinkImage img="ico-pause.svg" desc="아이콘" /> to switch from real-time mode to non-real-time mode.

2. In <Cmdname sid="time" className="uitext" />, set the time zone to view.

3. On the upper right of the <Cmdname sid="active_sessions" className="uitext" /> area, select <Cmdname sid="TTL06468" className="uitext" />.

<Cmdname sid="TTL06468" className="uitext" /> appears in a new window. In <Cmdname sid="instance_monitoring" className="uitext" />, you can see the top SQL list executed for the set time. For more information about <Cmdname sid="TTL06468" className="uitext" />, see [the following](stat).
