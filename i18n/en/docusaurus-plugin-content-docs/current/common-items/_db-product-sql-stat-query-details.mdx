If you select query from the search result list, the **_SQL details_** window appears. 

-   **_View plans_**: If you enter the DB name, user name, and password, you can see the **_plan_** information.

-   **_Parameters_**: The part set as a variable in the query sentence can be bound to the **_default_** value. If you select **_clear_**, the bound value is released.

-   **_Formatting_**: If you select the toggle button, the SQL statement can be indented and reformatted to improve readability.

-   **_Associated applications_**: You can go to the application project associated with the query. Select a project in the list and then click ![Arrow icon](/img/ic-arrow-right.svg). Go to **_STAT/REPORT_** > **_Statistics_** under the selected project.
