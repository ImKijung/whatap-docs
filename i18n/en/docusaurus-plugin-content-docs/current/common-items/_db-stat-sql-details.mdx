When the bar graph is selected in the chart, the <Cmdname sid="session_detail" className="uitext" /> window appears where you can check statistical data for 5 minutes. The list of SQL executions appears for the selected time. When an item is selected, it goes to the <Cmdname sid="instance_monitoring" className="uitext" /> menu with a new window.

In <Cmdname sid="instance_monitoring" className="uitext" />, you can check active sessions in the table and trends of various metrics in the selected time zone.
