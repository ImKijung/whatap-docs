While you can monitor only one instance in <Link to="instance-monitoring"><Cmdname sid="side_dbMonitoring" className="uitext" /></Link>, <Cmdname sid="side_multiInstance" className="uitext" /> allows you to see and compare the metrics for multiple instances on a screen. You can quickly see the status of instances through grouped values of instance metrics such as running agent count, total number of active sessions, and transaction information. Graph charts make it easy to see how instances' metric values are trending and which instances have higher metric values. It provides information of the longest lasting active session.
