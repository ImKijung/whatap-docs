In **_[Instance Monitoring](instance-monitoring)_**, to monitor only one instance, **_Multiple Instance Monitoring_** allows you to check and compare metrics of multiple instances on one screen. You can quickly see the status of instances through grouped values of instance metrics such as running agent count, total number of active sessions, and transaction information. Graph charts make it easy to see how instances' metric values are trending and which instances have higher metric values. It provides information of the longest lasting active session.
