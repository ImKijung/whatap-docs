## Copying the Flex board {#copyflexboard}

You can copy the created Flex board and then deliver it to other members.

In <Cmdname sid="dashboard_list" className="uitext" /> under <Cmdname sid="integratedFlexboard" className="uitext" />, select <LinkImage img="ico-copy.svg" className="ico" desc="Copy icon" />. Select <Cmdname sid="copy_to_me" className="uitext" /> or <Cmdname sid="copy_to_others" className="uitext" /> option.

- If you select the <Cmdname sid="copy_to_me" className="uitext" /> option, the items with the suffix "_copied" are added to <Cmdname sid="dashboard_list" className="uitext" />.

- If <Cmdname sid="copy_to_others" className="uitext" /> is selected, click <LinkImage img="ico-add.svg" className="ico" /> <Cmdname sid="account" className="uitext" />.

  <ImgLang img='flexboard-copy2others.png' desc='Copy to others' />

  Select another member's email address and then click <Cmdname sid="copy_dashboard" className="uitext" />. The items with the suffix "\_copied" are added to the <Cmdname sid="dashboard_list" className="uitext" /> list for copied members.

## Sharing as a JSON file

You can save the Flex board settings as a JSON file and pass them to another member or import the settings of the member.

### Export

1. Go to <Cmdname sid="integratedFlexboard" className="uitext" /> or <Cmdname sid="flexboard" className="uitext" /> for the project.

2. In <Cmdname sid="dashboard_list" className="uitext" />, select <LinkImage img="ico-export.svg" desc="Export icon" className="ico" />.

3. Once the JSON file has been downloaded, forward it to other members for sharing.

### Import

1. Go to <Cmdname sid="integratedFlexboard" className="uitext" /> or <Cmdname sid="flexboard" className="uitext" /> for the project.

2. On the upper right of the screen, select ![Import icon](/img/ico-import.svg) <Cmdname sid="import" className="uitext" />.

3. Select a JSON file to download.

:::note

This feature is available only to the members who have the **Edit** role on the project.

:::
