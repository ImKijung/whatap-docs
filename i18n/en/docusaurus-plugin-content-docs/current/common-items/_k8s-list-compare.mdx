By using the **_Compare_** option, you can see the comparison chart for key metrics of multiple <InDoc pages="resource-container-list">containers</InDoc><InDoc pages="resource-node-list">nodes</InDoc>.

1. If you select the ![number a](/img/char-a.svg) **_Compare_** option, the comparison column is added in the <InDoc pages="resource-container-list">container</InDoc><InDoc pages="resource-node-list">node</InDoc> list.

2. Through the <LinkImage img="char-b.svg" className="ico" /> comparison column on the left of the <InDoc pages="resource-container-list">container</InDoc><InDoc pages="resource-node-list">node</InDoc> list, select a <InDoc pages="resource-container-list">container</InDoc><InDoc pages="resource-node-list">node</InDoc> to compare.

   - You can select up to 10 <InDoc pages="resource-container-list">containers</InDoc><InDoc pages="resource-node-list">node</InDoc>.

   - Selected <InDoc pages="resource-container-list">containers</InDoc><InDoc pages="resource-node-list">nodes</InDoc> are given different colors.

3. You can see the comparison chart for the selected <InDoc pages="resource-container-list">containers</InDoc><InDoc pages="resource-node-list">nodes</InDoc> in the ![number c](/img/char-c.svg) area. 

   - You can see the <InDoc pages="resource-container-list">containers</InDoc><InDoc pages="resource-node-list">nodes</InDoc> selected in the ![number b](/img/char-b.svg) comparison column at the top of the ![number c](/img/char-c.svg) area.

   - By dragging the top of ![number c](/img/char-c.svg), you can adjust the height of the comparison chart area.

4. If you select ![number d](/img/char-d.svg) **_Check period_**, the comparison chart appears withing the period to view as follows:
