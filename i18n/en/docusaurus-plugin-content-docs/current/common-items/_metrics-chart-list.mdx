:::note 

**Data Merge**

Data merge provides the following methods: <Cmdname sid="object_merge" className="uitext" /> and <Cmdname sid="time_merge" className="uitext" />. 

- <Cmdname sid="time_merge" className="uitext" /> merges data with the same field values at regular intervals in the original data.

- <Cmdname sid="object_merge" className="uitext" /> merges data with matching tags among data with different field values.

:::
