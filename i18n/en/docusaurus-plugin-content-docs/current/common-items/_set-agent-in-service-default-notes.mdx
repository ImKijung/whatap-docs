:::note

- If you select <Cmdname sid="okind_default" className="uitext" /> or <Cmdname sid="project_default" className="uitext" /> tab and then apply the options, it is useful to commonly manage agents.

- The application order of options is as follows: <Cmdname sid="okind_default" className="uitext" />, <Cmdname sid="project_default" className="uitext" />, and <Cmdname sid="agent_configuration" className="uitext" />. 

  - If the value of option b is 1000 in <Cmdname sid="project_default" className="uitext" /> and it is 2000 in <Cmdname sid="okind_default" className="uitext" />, the option value set in <Cmdname sid="okind_default" className="uitext" />, 2000 takes precedence.
  - If the value of option b is null in <Cmdname sid="project_default" className="uitext" /> and it is 2000 in <Cmdname sid="okind_default" className="uitext" />, the option value set in <Cmdname sid="okind_default" className="uitext" /> takes precedence.

:::
