### Modifying or deleting options

<ProdImg img='modify-agent.png' desc='Modifying options' />

1. Scroll up or down the screen, or select a desired option to edit or delete from the option list on the left. 

2. Select or edit the desired value for the option to change. To delete an option, select ![Delete icon](/img/ico-trash.svg). 

3. Select <Cmdname sid="apply" className="uitext" /> to apply your changes.

:::note

- If you want to move to the top of the screen, select ![number3](/img/number-03.png) <Cmdname sid="go_to_add_option" className="uitext" />.

- If you select <Cmdname sid="apply" className="uitext" /> without entering options, the options are deleted.

:::
