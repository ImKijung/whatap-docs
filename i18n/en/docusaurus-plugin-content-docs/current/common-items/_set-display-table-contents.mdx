1. Select ![Configuration icon](/img/ico-setting.svg) on the upper right of the table.

2. From the drop-down menu, select <Cmdname sid="table_content_setting" className="uitext" />.

3. Set the following option:

   - <Cmdname sid="whether_display_content" className="uitext" />: Select whether or not to display the content displayed in the table list.

   - <Cmdname sid="tag_management" className="uitext" />: You can select the desired tags to display in the table list. You can make them invisible or change their order.

4. After all settings are finished, select ![Close icon](/img/ico-close.svg).

The set content is applied to the table.
