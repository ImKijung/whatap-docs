-   <Cmdname sid="time_selector" className="uitext" />

    -   You can select the lookup time by selecting the green button on the right.

    -   By selecting ![Left arrow icon](/img/left-arrow.svg) or ![Right arrow icon](/img/right-arrow.svg), you can move by the selected inquiry time range.

    -   To select a detailed time, select a date or time zone. Set the detailed time and then select <Cmdname sid="apply" className="uitext" />.

-   Based on <Cmdname sid="filter" className="uitext" />: <Cmdname sid="title" className="uitext" /> or <Cmdname sid="message" className="uitext" />, you can filter event records.

-   <Cmdname sid="all_selected" className="uitext" />: You can select an agent included in the project.

-   **_CSV_**: You can save the viewed event records as a CSV file. Enter the <Cmdname sid="maximum_csv_lines" className="uitext" /> and then select <Cmdname sid="download" className="uitext" />.

-   <Cmdname sid="column_select" className="uitext" />: You can add columns to the viewed event records.

-   <Cmdname sid="event_setting" className="uitext" />: Goes to <Cmdname sid="event_statistic" className="uitext" /> > <Cmdname sid="event_setting" className="uitext" />.
