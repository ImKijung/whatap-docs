### Adding the event reception tag

1. In the user list of the <Cmdname sid="alert_receive_settings_by_member" className="uitext" /> section, select <Cmdname sid="add_tag" className="uitext" /> or <LinkImage img="ico-add.svg" desc="add icon" />.

2. If the <Cmdname sid="event_tag" className="uitext" /> pop-up window appears, enter the tag name in the tag field and then press Enter or select <Cmdname sid="create_new_eventtag" className="uitext" />.

3. Select the tag color.

4. Select <Cmdname sid="create_eventTag" className="uitext" />.

In <Cmdname sid="event_tag_list" className="uitext" />, you can see the created tags. To apply any created tag, select the tag.
