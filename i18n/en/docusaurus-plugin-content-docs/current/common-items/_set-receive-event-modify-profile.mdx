## Setting the alert user

In <Cmdname sid="MNU07360" className="uitext" />, you can configure the settings for user's individual notification level, reception method, day of the week, and time.

1. Select the profile icon on the upper right of the screen.

2. If a pop-up menu appears, select <Cmdname sid="MNU07360" className="uitext" />.

3. Scroll down the screen, go to the <Cmdname sid="notification_setting" className="uitext" /> section.

4. Set the reception level, reception method, day of the week, and time, and then select <Cmdname sid="save" className="uitext" />.
