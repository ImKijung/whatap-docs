### Receiving notifications when event reception tags are not set

<ImgLang img='receive-untag-event.png' desc='수신 태그' />

To receive alerts with no <Cmdname sid="event_tag" className="uitext" /> setting, select the <Cmdname sid="ITM05991" className="uitext" /> option. To receive only the alerts with the <Cmdname sid="event_tag" className="uitext" /> setting, uncheck the option.

:::note

If you do not want to receive all alerts, uncheck the option and there must be no selected <Cmdname sid="event_tag" className="uitext" />.

:::
