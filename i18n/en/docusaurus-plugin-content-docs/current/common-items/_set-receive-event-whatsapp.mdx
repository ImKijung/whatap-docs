### Adding to receive WhatsApp notifications

The notifications can be received through WhatsApp.

<ImgLang img='set-event-whatsapp-notification.png' desc='WhatsApp'/>

1.  Select the profile icon on the upper right of the screen. 

2.  If a pop-up menu appears, select **_Manage Account INFO._**. 

3.  In the **_User phone_** section, select **_WhatsApp_**. 

4.  In **_Phone_**, enter the phone number to receive SMS.

5.  Select **_Send your credentials_**.

6.  Enter the 6-digit verification code sent through your WhatsApp application.

7.  Select **_Verify_**.

:::note

To change the registered phone number, select **_Change the Number_** and then repeat the above steps.

:::
