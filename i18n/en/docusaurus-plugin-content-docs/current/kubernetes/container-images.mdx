This document provides information about the <Cmdname sid="containerImages" className="uitext" /> menu. The <Cmdname sid="containerImages" className="uitext" /> menu provides available images in the containers, number of containers running with those images, and the total usage amounts for CPU and memory in the list format.

![Container image](/img/k8s-container-img.png)

### ![](/img/char-a.svg) Filter

If <Cmdname sid="TTL06740" className="uitext" /> is selected in the Filter window, the filtering for **_Image Name_**, **_Image Tag_**, and **_Repo Digest_** is provided.

### ![](/img/char-b.svg) Download CSV

By selecting <Cmdname sid="sms_summary_download_csv" className="uitext" /> in the upper right corner, you can download the original data as a file in CSV format. 

> File name example: `kubernetes_container_images_{pcode}_{timestamp}.csv`

### ![](/img/char-c.svg) Container details view

If you select the ![Display Detail icon](/img/ic-pageview.svg) <Cmdname sid="TLT07247" className="uitext" /> icon on the right of the **_Image Name_** column, the <Cmdname sid="container_list" className="uitext" /> menu appears where the image has been searched.
