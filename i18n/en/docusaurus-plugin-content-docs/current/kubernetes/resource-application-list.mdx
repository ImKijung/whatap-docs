Home > Select Project > **_Workload_** > **_Container Application List_**

It displays the list of containerized applications. If you select a desired column, you can search the application list sorted by that column. 

<ImgLang img='k8s-app-list.png' desc='Application List' />

- Real-time data is updated every 10 seconds.

- It provides the services for searching, column sorting, and detailed view, and _CSV_ file downloading.

  > CSV file name, _kubernetes_application_list\_`{pcode}`\_`{timestamp}`.csv_
