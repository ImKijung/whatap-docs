Before starting the Kubernetes monitoring, check out the following supports. WhaTap Kubernetes can be installed on standard Kubernetes versions 1.16 to 1.28 and the equivalent Kubernetes environments.

## Supported platforms

-   Amazon EKS (Elastic Kubernetes Service)

-   AKS (Azure Kubernetes Service)

-   GKE (Google Kubernetes Engine)

-   OKE (Oracle Container Infrastructure Container Engine for Kubernetes)

-   Naver Ncloud Kubernetes Service

-   NKS (NHN Kubernetes Service)

-   Red Hat OpenShift (3.xx / 4.xx)

-   Cocktail

-   Accordion

-   PaaS-TA

-   KaKao i Cloud

-   Rancher

-   Plug-ins: Supports NVIDIA GPU
