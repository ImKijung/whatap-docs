It guides you to the elements displayed on the initial screen, after you log in [WhaTap Monitoring Service](https://service.whatap.io). The default menu of the initial screen is <Cmdname sid="project_list" className="uitext" />.

<ImgLang img='main-ui.png' desc='초기 화면' />

## ![](/img/number-01.png) Fixed menu area

-   **Menu**: The menu path appears on the left.

-   ![Notice icon](/img/ic-notice.svg) <Cmdname sid="TTL06763" className="b600" />: You can see the notices of the WhaTap monitoring service.

-   ![This page theme icon](/img/ico-theme.svg) <Cmdname sid="theme" className="b600" />: You can change the theme of the current page. The default value is **_Light_** mode.

    -   **_Light_**: Theme that is applied by default.
    -   **_Dark_**: The theme with the black background is applied.
    -   **_Presentation_**: The optimized color is applied when using a beam projector.
    -   **_Add Custom Theme_**: You can set the desired color.

-   ![WhaTap Docs icon](/img/ic-guide.svg) **WhaTap Docs**

    Go to the WhaTap Docs technical documentation page. For more information about Whatap Docs, see [the following](welcome-to-whatapdocs).

-   ![Customer Support icon](/img/ico-help.svg) **Customer Support**

    Online customer support is available. Select **_Start a chat_** and then send your questions about technical and payment issues. For previous records, select **Message**. To make a wired inquiry, select ![Wired Connection icon](/img/ico-tel.svg).

    To set the inquiry language, notification, or advertisement reception, select ![Setting and Information icon](/img/ico-setting.svg).

-   **Profile icon**: Select the rightmost icon. It provides <Cmdname sid="manage_account_info" className="uitext" />, <Cmdname sid="create_organization" className="uitext" />, and <Cmdname sid="organization_management" className="uitext" />. In addition, it also provides the <Cmdname sid="language" className="uitext" /> function to select a user language. To log out of the current service screen, select <Cmdname sid="log_out" className="uitext" />.

## ![](/img/number-02.png) Project search area

-   <Cmdname sid="placeholder_input_pcode_or_project_name" className="uitext" /> (text entry field): You can enter the project name or code to search for the desired project.

    :::note

    The project code (**_PCODE_**) can be seen in <Cmdname sid="pcode" className="uitext" /> under the <Cmdname sid="management" className="uitext" /> > <Cmdname sid="project_management" className="uitext" /> menu of the project.

    :::

-   ![Favorite icon](/img/ico-favorite.svg) <Cmdname sid="project_favorite" className="uitext" />: Only the projects set as favorite appear in the list. To set the project as favorite, select ![Favorite icon](/img/ico-favorite.svg) at the upper left of the project card view in the project list. To cancel the favorite setting, select ![Favorite icon](/img/ico-favorite-set.svg) again.

-   ![Wide card view icon](/img/ico-extend-card-view.svg) <Cmdname sid="wide_card_view" className="uitext" />: You can see the project list in the form of a card. The basic project information is also displayed.

-   ![Compact card view Icon](/img/ico-reduction-view.svg) <Cmdname sid="compact_card_view" className="uitext" />: You can see the project list in the form of compact card rather than <Cmdname sid="wide_card_view" className="uitext" />.

-   ![List View icon](/img/ico-list-view.svg) <Cmdname sid="project_list_view" className="uitext" />: You can see the project list in the form of table by classifying the projects by product catalog.

-   **_15 / page_**: You can set the number of projects to be displayed for each page. You can set up to 100 projects to be displayed on a page.  To go to another page, select the page number.

## ![](/img/number-03.png) Organization and group, project area

-   <Cmdname sid="sidebar_org_selector_no_organization" className="uitext" />: You can select an organization or go to <Cmdname sid="organization_management" className="uitext" />. For more information about organization management, see <Link to="project/organization">the following</Link>.

-   **_+_** <Cmdname sid="BTN06697" className="uitext" />: You can add a group. For more information about group addition, see [the following](project/group).

-   **_+_** <Cmdname sid="BTN06696" className="uitext" />: You can add a new project. For more information about project addition, see [the following](project/project-manage#create-project).

-   ![Search icon](/img/ico-search.svg) <Cmdname sid="search" className="uitext" />: You can perform search by the group, project, or agent. For more information about project search, see [the following](#search-project).

-   ![Filter icon](/img/ico-filter.svg) <Cmdname sid="project" className="uitext" /> <Cmdname sid="type" className="uitext" /> <Cmdname sid="filter" className="uitext" />: By setting filters, you can see the project list by product category. It displays a list under <Cmdname sid="all_projects" className="uitext" /> according to the set value.

    <ImgLang img='main-project-filter.png' desc='프로젝트 목록 필터링' />

-   If you select ![Up arrow icon](/img/up-arrow.svg) or ![Down arrow icon](/img/down-arrow.svg), you can show or hide the project list area.

-   <Cmdname sid="all_projects" className="uitext" />: You can see the list of all projects. If you select this menu after entering a project, you can go to the initial project list screen.

-   Under the area, the project list is displayed. The group is displayed as the ![Folder icon](/img/ico-group.svg) shape. Non-group projects appear with icons for each product type.

### Searching for a project{#search-project}

1.  If you select ![Search icon](/img/ico-search.svg) on the upper left of the screen, the project <Cmdname sid="search" className="uitext" /> window appears.

    <ImgLang img='search-project.png' desc='프로젝트 검색 창' />

2.  Enter a project or group name. Then the matched project list appears under the search window. 

3.  Select a desired project.

The corresponding project screen appears.

:::note

To search for a project based on the agent in <Cmdname sid="search" className="uitext" /> window, click <Cmdname sid="group_or_project" className="uitext" /> and then select <span class="uitext">Agent</span> from the drop-down list.

:::

## ![](/img/number-05.png) Project list

You can check the list of projects in which the user has been registered as a member.

## ![](/img/number-06.png) Menu area at the bottom

-   <Cmdname sid="my_usage" className="uitext" />: You can use the menus related to the payment information, usage pricing, etc. For more information, see <Link to="management/billing">the following</Link>.

-   ![Menu Fold icon](/img/ic-menu-fold.svg): You can hide or display the side menu. When the side menu is hidden, select ![Menu Expand icon](/img/ic-menu-expand.svg) to display the side menu.

-   ![Full screen icon](/img/ic-fullscreen.svg): You can use the WhaTap monitoring service in full screen. To end the full screen, select the ESC key.
