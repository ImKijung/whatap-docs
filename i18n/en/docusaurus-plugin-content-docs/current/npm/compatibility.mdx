와탭의 **NPM**을 사용하기 위해서는 운영 중인 서버들에 **NPM Agent**를 설치해야 합니다. **NPM**은 OS 커널에서 제공하는 eBPF 기술을 사용하고 있기 때문에 OS 버전 확인이 필요합니다.

-   Linux 커널 5.4 버전 이상을 지원합니다.

-   Linux 배포판 기준 Ubuntu 20.04, CentOS Stream 9 이상을 권장합니다.

:::note

-   Ubuntu, CentOS의 과거 버전에서도 커널 업데이트를 통해 사용 가능합니다.

-   그 외의 다른 배포판에서도 커널 버전이 맞으면 사용 가능하지만 현재 공식적으로 테스트된 배포판은 Ubuntu와 CentOS입니다.

-   현재 Windows, macOS는 지원하지 않습니다.

:::
