## Downloading the WhaTap database agent

{@include: ../common-items/_db-install-agent-steps.mdx} 

3. Enter the unzipped folder and then check the _whatap.conf_ file. In _whatap.conf_, enter the project access key(`license`), WhaTap server data(`whatap.server.host`), and DB connection data(`dbms`, `db_ip`, `db_port`).

   ```ini title='whatap.conf'
   license={Access_Key}
   whatap.server.host=13.124.11.223/13.209.172.35 # WhaTap server information

   dbms=redis
   db_ip={DB_server_IP_address}
   db_port={DB_server_port}
   ```
