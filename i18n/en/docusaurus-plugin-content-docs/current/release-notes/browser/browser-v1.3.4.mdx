February 19, 2024

- <Status>Feature</Status> Added the function to collect the following metrics in <b>Network Information API</b> upon occurrence of page load or AJAX events.

  - downlink
  - effectiveType
  - rtt

- <Status>Feature</Status> Added the function to collect the following metrics upon occurrence of resource events.

  - deliveryType
  - responseStatus

- <Status>Feature</Status> Added the function to collect Content Security Policy (CSP) errors.

:::note

For more information about the added metrics, see the following:

- [<Cmdname sid="TTL07611" className="uitext" />](https://developer.mozilla.org/en-US/docs/Web/API/NetworkInformation/effectiveType) (effectiveType)
- [<Cmdname sid="TTL07612" className="uitext" />](https://developer.mozilla.org/en-US/docs/Web/API/NetworkInformation/rtt) (rtt)
- [<Cmdname sid="TTL07610" className="uitext" />](https://developer.mozilla.org/en-US/docs/Web/API/NetworkInformation/downlink) (downlink)
- [<span class="uitext">Delivery Type</span>](https://developer.mozilla.org/en-US/docs/Web/API/PerformanceResourceTiming/deliveryType) (deliveryType)
- [<Cmdname sid="status_code" className="uitext" />](https://developer.mozilla.org/en-US/docs/Web/API/PerformanceResourceTiming/responseStatus) (responseStatus)

:::
