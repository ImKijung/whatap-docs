October 4, 2023

## Common

<p><Status>Fixed</Status> Fixed the bug where the `invalid parameter key` message appears even when the session kill function is working normally.</p>

## PostgreSQL

- <Status>Fixed</Status> Modified the deadtuple collection query sentence.

- <Status>Fixed</Status> Modified not to create 5-minute statistical data for the categories with the collection interval of 1 day.

## MySQL

- <Status>Fixed</Status> Fixed the bug where the table size is not collected in the Maria DB.

- <Status>Fixed</Status> Modified the active session's state value from <code>S</code> to <code>Sending data</code>.

## Oracle

<p><Status>New</Status> Added various ratio metrics. Set the option as follows:</p>

```ini title='whatap.conf'
ora_ratio=true
```

The metrics to collect are as follows:

- `buffer cache hit ratio`
- `soft parse ratio`
- `hard parse ratio`
- `library cache hit ratio`
- `library cache pin hit ratio`
- `library cache get hit ratio`
- `latch hit ratio`
- `shared pool free mem ratio`
