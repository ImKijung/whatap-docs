#### August 16, 2023

- <Status>Change</Status> Change of the application pool selection method

  - Before: Filtering to prevent from operating while the monitoring module is loaded into memory

  - Change

    - The monitoring module is not loaded unless it is for the target application pool.

    - No impact is given on the customer's .NET application because it does nothing for monitoring.

    - Changed the condition to restart IIS when any settings are changed.

- <Status>Change</Status> Downgraded the .NET version of the server support environment analysis tool to 4.61.
