November 3, 2023

-   <Status>Fixed</Status> Fixed the issue where the CPU utilization increased in the recent version of agent.

    :::info

    The system API of the Go library that loads the current time requires a lot of CPU performance, so it has been improved efficiently.

    :::

-   <Status>Fixed</Status> Fixed the bug where unknown steps appear additionally in case of WCF transactions.

-   <Status>Fixed</Status> Fixed the bug where the previous installation is not deleted during an update installation.

-   <Status>Changed</Status> Freed cached memory as quickly as possible to reduce memory occupancy.
