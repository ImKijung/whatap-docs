November 10, 2023

- <Status>Fixed</Status> Fixed the error that occurs when SQL trace data has exceeded 32 KB and connection data has been longer than the SQL string, and added exception handling for all code that may cause similar string-related errors.

- <Status>New</Status> Improved packet-related memory management to lower the memory and CPU usage.
