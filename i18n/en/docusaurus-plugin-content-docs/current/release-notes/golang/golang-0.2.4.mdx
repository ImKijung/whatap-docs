November 22, 2023

- <Status>Changed</Status> Changed the text cache storage interval from one day to one hour.

- <Status>Changed</Status> Changed not to collect CPU and memory metrics in the Kubernetes environment, replacing them with system metrics collected by the Kubernetes agent.

- <Status>Fixed</Status> Fix the whatap-agent service execution error in CentOS 9 or later.

- <Status>Fixed</Status> Fixed the duplicate collection of vCpu information in the Docker container environment

- <Status>Fixed</Status> Fixed the error where ports are collected together while collecting client IP addresses.

:::info
