January 10, 2024

- <Status>Feature</Status> Trace context (W3C) is supported in the multi-transaction tracing.

  **OpenTelmetry** that supports the trace context, and the multi-transaction tracing correlation are supported.

- <Status>Changed</Status> Modified the resource collection for SQL, Method, and HTTP call.

  The differences between resources (CPU, memory) at the start of the transaction and the start of each step are collected.

- <Status>Changed</Status> Modified to newly create the configuration file (<em>whatap.conf</em>) if nothing.

- <Status>Fixed</Status> Fixed the error where the cumulative number of outbound calls are collected wrong as the cumulative time in the active transation list.

- <Status>Fixed</Status> Fixed the whatap-agent service (<code>whatap-agent.service</code>) execution error in the Linux environment.

:::info
