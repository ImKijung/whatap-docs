September 26, 2023

- <Status>Feature</Status> By setting the super of HttpServlet, the function to start an HTTP service has been added in the class that inherits the class. You can use comma (,) as the delimiter.

  ```ini title='whatap.conf'
  # example, separator ','
  hook_httpservlet_supers=javax.servlet.http.HttpServlet
  ```

- <Status>Feature</Status> db2-11.5 tracing

  ```ini title='whatap.conf'
  weaving=db2-11.5
  ```

- <Status>Feature</Status> Oracle database trace settings have been added. Restarting is required after applying the option. The default value is `true`.

  ```ini title='whatap.conf'
  # Default: true
  # Will require a restart of the agent to take effect.
  hook_oracle_enabled=true
  ```

- <Status>Changed</Status> During lettuce-5.1 tracing, modified to display key values for the collection SQL.

- <Status>Fixed</Status> Modified to delete logs generated when collecting container IDs.

- <Status>Fixed</Status> Fixed the issue where SQL statements do not appear while tracing the Redis lettuce driver (<b>lettuce-5.1</b>, <b>lettuce-6.2</b>).

- <Status>Deprecated</Status> While tracing HTTP services, excluded the <code>org.apache.catalina.filters.FilterBase</code> class from the default tracing.
