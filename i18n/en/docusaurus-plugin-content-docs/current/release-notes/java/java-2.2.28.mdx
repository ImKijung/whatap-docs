February 27, 2024

- <Status>Feature</Status> When tracing the logback-1.2.8, added the appender exclusion option.

  ```ini title="whatap.conf"
  # example: ignore 'CONSOLE', 'ACCESS_APPENDER' appender
  ignore_logback_appender_set=CONSOLE,ACCESS_APPENDER
  ```

- <Status>Deprecated</Status> Stopped using the option to collect only the specified appenders while tracing the logback-1.2.8.

  ```ini title="whatap.conf"
  # deprecated
  trace_logback_appender_set=
  ```

- <Status>Feature</Status> When tracing the log4j-2.17, added the appender exclusion option.

  ```ini title="whatap.conf"
  # example: ignore 'CONSOLE', 'ACCESS_APPENDER' appender
  ignore_log4j_appender_set=CONSOLE,ACCESS_APPENDER
  ```

- <Status>Deprecated</Status> Stopped using the option to collect only the specified appenders while tracing the log4j-2.17..

  ```ini title="whatap.conf"
  # deprecated
  trace_log4j_appender_set=
  ```

- <Status>Changed</Status> If the Pod name has not been specified in the container environment, modified to change it to be extracted from <code>HOSTNAME</code> in the environment variable (<code>env</code>).

- <Status>Changed</Status> Modified to limit the number of logs when connected to JBoss Connection Pool.

- <Status>Changed</Status> Modified the logs when sending the statistical data.

- <Status>Fixed</Status> Fixed the excessive TPS collection issue when using the <code>trace_sampling_enabled</code> option.

- <Status>Fixed</Status> Fixed the error that occurs when collecting the spring-security exception for spring-boot-3.x.
