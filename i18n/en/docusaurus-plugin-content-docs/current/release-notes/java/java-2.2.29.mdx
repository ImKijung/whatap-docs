March 15, 2024

- <Status>New</Status> Added the jboss-logging-3.3 log collection.

  ```ini title='whatap.conf'
  weaving=jboss-logging-3.3
  ```

- <Status>Feature</Status> Added the SQL data collection when using the <code>com.mysql.cj.jdbc.ClientPreparedStatement.executeBatchInternal</code> method.

- <Status>Fixed</Status> Prevented the issue where a transaction is not completed after a bug occurs when collecting HTTP parameters in the Jboss environment.

- <Status>Deprecated</Status> Stopped collecting the <b>HOSTNAME</b> for <code>env</code>.
