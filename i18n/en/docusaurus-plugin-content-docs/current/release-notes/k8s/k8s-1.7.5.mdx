June 18, 2024

<p><Status>Feature</Status> Added the metrics collection option for the disks with the nfs file system.</p>

- `collect_nfs_disk_enabled`: This option determines whether to collect the nfs disks. The default value is `true`. 

:::note 

This option can be set through the whatap-node-helper container environment variable.

:::
