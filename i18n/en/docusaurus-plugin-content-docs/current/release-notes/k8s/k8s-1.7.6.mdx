July 1, 2024

- <Status>Feature</Status> Added the tag count.

  - Added **kube_service_pod_mapping**.
  - Added the `podUid` tag in the **kube_pod** category.
  - Added the `externalIp` tag and PORT(`portName`, `nodePort`, `servicePort`, `targetPort`) field in the **kube_service** category.
  - Added the data for `watch_event_type` and `resource_version` in the **kube_event** category.
  - Removed duplicate **kube_event** notifications.

- <Status>Feature</Status> Added `WHATAP_JAVA_AGENT_PATH` in the **whatap-node-helper** container environment variable (`env`).

  :::note

  - You can link the container ID by setting the WhaTap Java Agent's jar file path in the corresponding environment variable. 

  - This function is supported in [Java agent 2.2.33](../java/java-2_2_33) or later.

  :::
