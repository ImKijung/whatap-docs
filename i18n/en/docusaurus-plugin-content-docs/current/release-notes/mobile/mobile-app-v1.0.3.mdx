December 19, 2023

The WhaTap's mobile app has been newly released.

- On the mobile app, you can see the main charts of the dashboard that are viewed through the WhaTap monitoring service.

- Data can now be viewed and displayed more quickly. 

- The OTP function can be used as a secondary authentication tool when logged in. 

- An organization feature has been added so that you can see the projects in it.

- You can see the received alerts in more detail. 

- English and Japanese are supported.
