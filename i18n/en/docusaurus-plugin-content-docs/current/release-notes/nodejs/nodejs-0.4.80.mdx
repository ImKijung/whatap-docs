October 31, 2023

-   <Status>Changed</Status> Modified the Datatext agent information delivery time from 1 day to 1 hour.

-   <Status>Fixed</Status> Fixed the error to deliver values for <code>ONAME</code> and <code>ONODE_NAME</code> in the Kubernetes environment.

-   <Status>Fixed</Status> Modified to apply the setting when the originUrl option (<code>whatap_micro_enabled</code>) is changed in the config file.

-   <Status>Fixed</Status> Fixed the issue where query tracking is not performed upon application of replication in the Typeorm and MySQL environments.

-   <Status>Fixed</Status> Fixed the issue where tracking is not possible when query is made using Redis.

:::info

-   Package information: https://www.npmjs.com/package/whatap

-   Installation and application methods

    ```bash
    npm i whatap
    ```

:::
