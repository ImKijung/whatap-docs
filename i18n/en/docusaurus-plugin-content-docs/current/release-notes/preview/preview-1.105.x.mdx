## Preview 1.105.05

2023년 7월 5일

<div class='indentTab'>

### `Common`

<p><Status>Change</Status> 비밀번호 유효성 체크 로직 변경</p>

-   최소 9자리의 영어 소문자 및 대문자, 숫자, 특수 문자 중 3종류 이상

-   연속된 숫자가 포함된 비밀번호 사용 불가

-   현재 비밀번호와 동일한 비밀번호 사용 불가
