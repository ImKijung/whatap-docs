## Preview 1.105.05

July 5, 2023

<div class='indentTab'>

### `Common`

<p><Status>Change</Status> Modified the password validation logic.</p>

-   Use at least 9 characters with 3 combinations among English lowercase and uppercase letters, numbers, and special characters.

-   The same password as your email cannot be used.

-   The same password as the one you used cannot be used.

-   A password containing consecutive keyboard arrangements is not allowed.

-   A password containing consecutive alphabets is not allowed.

-   A password containing consecutive numbers is not allowed.

-   The same password as the current one cannot be used.

* * *

### `Container`

<p><Status>Change</Status> In <UI>Log</UI> > <UI>Log setting</UI>, modified the node agent log setting so that it can be applied with a click instead of directly entering options for user convenience.</p>

* * *

### `URL`

<p><Status>Fixed</Status> When using the <UI>Add URL</UI> function in <UI>URL list</UI>, deleted the duplicates in the <UI>Check location</UI> section.</p>

</div>
