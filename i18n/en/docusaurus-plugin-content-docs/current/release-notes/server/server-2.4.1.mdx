October 4, 2023

- <Status>Fixed</Status> Fixed the agent malfunction that occurs when entering the domain name in webcheck due to the executable image build option error.

- <Status>Changed</Status> In the <code>filesystem.encoding</code> option, added <code>shift_jis</code> and <code>euc_jp</code> for support of multilingual log files.

:::info
