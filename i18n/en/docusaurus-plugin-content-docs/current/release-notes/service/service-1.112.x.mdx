### `Application`

-   <Status>Changed</Status> Unified and standardized the names of metrics used repeatedly.

    -   URL, Transaction, Transaction name → Transaction
    -   Domain, 도메인 → Domain
    -   DBC Time, DB CONN. time, DB connect time → DB CONN. time
    -   URL hash → Transaction Hash
    -   CPU time, CPU usage time → CPU usage time
    -   Memory allocation, Memory allocation usage → Memory Allocation
    -   HTTPC, HTTP call, HTTPC count → HTTP call count
    -   Performed standardization by establishing writing rules for multiple overlapping terms.

-   <Status>Changed</Status> In <Cmdname sid="stat_report_upper" className="uitext" /> > <Cmdname sid="statistics" className="uitext" />, modified to maintain the search time set for tab switching.
