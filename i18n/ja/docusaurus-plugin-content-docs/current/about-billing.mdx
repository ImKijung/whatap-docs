WhaTapモニタリングサービスの決済情報登録と決済権限移譲、プロジェクト有料転換など決済アカウント管理方法について案内します。 無料利用期間および無料利用基準超過時点でサービス利用を継続したい場合は、決済情報を登録し、該当プロジェクトを有料に転換しなければなりません。 

## ![](/img/number-01.png)決済情報登録

決済情報登録に先立ち、**決済**権限を確認してください。 基本的にプロジェクト作成アカウントに**決済**権限が付与されます。

1.  [WhaTapモニタリングサービス](https://service.whatap.io)にログインしてください。 

2.  ホーム画面左下で**_利用履歴_**を選択してください。

    <ImgLang img='tax_invoice_1.png' desc='利用履歴' />

3.  **_決済情報_**メニューから**_決済情報の登録_**ボタンをクリックしてください。

4.  **_決済情報の登録_**ウィンドウで決済情報をご記入ください。

5.  決済手段を選択し、**_クレジットカード_**定期決済または税金**_計算書発行_**関連情報を記入してください。

6.  **_請求書の受信者_**を設定してください。

7.  **_完了_**ボタンを選択してください。

:::note 

-   決済情報登録の詳細については、[次の文書](/management/billing)を参照してください。

-   多数対象に請求書を受け取るには、コンマ（,）を区切り文字で複数のメールを入力してください。 

    > 例、a@whatap.io,b@whatap.io,c@whatap.io

    <ImgLang img='best-p/about-billing-member-recipient.png' desc='請求先設定' />

:::

## ![](/img/number-02.png)決済権限移譲

**決済**権限は**1名**のみ持つことができます。 例えば、AアカウントからBアカウントに決済権限を移譲する場合、Aアカウントはこれ以上決済権限を保有しなくなります。 また、**決済**権限を移譲するアカウントが決済情報を登録した状態でなければ**決済**権限を移譲できません。

### 決済権限を移譲するアカウント決済情報登録

1.  [WhaTapモニタリングサービス](https://service.whatap.io)から**決済**権限を移譲するアカウントにログインしてください。

2.  ![指示線1](/img/number-01.png)のプロセスを通じて、そのアカウントの決済情報を登録してください。

### メンバー招待

1.  [WhaTapモニタリングサービス](https://service.whatap.io)で、現在**決済**権限および**修正**権限を保有しているアカウントでログインしてください。

2.  ホーム画面で決済権限を移譲するプロジェクトを選択してください。 

3.  **\*管理**メニュー下位**_プロジェクトメンバー_**メニューに移動してください。

4.  画面右上の**_メンバー招待_**ボタンを選択してください。

5.  メール入力欄に決済権限を移譲するアカウントのメールを入力し、**_追加_**ボタンをクリックしてください。

6.  **_招待メンバー一覧_**に該当アカウントが登録されたことを確認してから、**_完了_**ボタンを選択してください。

### 権限修正

<ImgLang img='best-p/about-billing-member-role.png' desc='権限削除' />

1.  **_プロジェクトメンバー_**一覧から決済権限を移譲するアカウントの右側に![指示線 a](/img/char-a.svg)![編集アイコン](/img/ico-edit.svg)修正ボタンを選択してください。

2.  **_メンバー権限の修正_** ウィンドウが表示されたら権限のうち、![指示線 b](/img/char-b.svg) **_決済_**権限を許可してください。

3.  設定を完了したら、![指示線 c](/img/char-c.svg) **_適用_**ボタンを選択してください。

:::note

-   決済権限を持つメンバーの場合、プロジェクトメンバー\*\*\*一覧で名前の右側に<code class='pay'>決済</code>タグが表示されます。 

-   **_メンバー権限の修正_**ウィンドウは、プロジェクトメンバー権限または組織およびグループメンバー権限の修正かによって異なります。 メンバー権限の詳細については、[次の文書](/project/project-structure#project-auth)を参照してください。

:::

## ![](/img/number-03.png) プロジェクト有償転換

決済情報登録完了時、**_プロジェクト有償転換_**メニューに移動します。 該当アカウントが保有しているすべての**_プロジェクト一覧_**を次のように確認することができます。 

<ImgLang img='best-p/about-billing-subscription.png' desc='プロジェクト有償転換' />

-   有償状態ではなく、プロジェクト全体を一括して有償転換する場合、上段右側の**_有償転換する_**ボタンをクリックしてください。

-   複数のプロジェクトを有償転換する場合は、**_プロジェクト一覧_**の各チェックボックスを選択し、上段右側の**_有償転換する_**ボタンをクリックしてください。

-   プロジェクト別の有償転換の場合、**_プロジェクト一覧_**の状態カラムで**_有償転換_**ボタンをクリックしてください。

-   **_プロジェクト一覧_**の各プロジェクト項目から**_有償転換_**ボタンをクリックし、個別的に有償転換することができます。

:::note

-   **_プロジェクト有償転換_**メニューの詳細は、[次の文書](management/billing#project-subscription)を参照してください。

:::
