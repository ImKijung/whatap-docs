ブラウザモニタリングで収集しているメトリクス指標の説明です。 **_メトリクス照会_**メニューから検索することができます。

## rum_page_load_each_page

ページロード時間に関する指標は、**PerformanceAPI**のうち**NavigationTiming**の値を参照します。 次の図のようにページロード探索が行われます。

![](https://w3c.github.io/navigation-timing/timestamp-diagram.svg)

ページロード応答時間は、**loadEventEnd**時点までを意味します。

-   収集間隔：5秒

-   統計データ：5分、1時間

### Tags

| タグ名          | 説明        | 備考 |
| ------------ | --------- | -- |
| `page_group` | ページグループ   | -  |

### Fields

<div class='browser-metric'>

| フィールド名                            | 単位  | 説明                                                              | 備考                                                                    |
| --------------------------------- | --- | --------------------------------------------------------------- | --------------------------------------------------------------------- |
| `page_load_backend_time`          | ミリ秒 | ページロード中のバックエンド領域の平均処理時間                                         | **startTime**から**responseEnd**までの所要時間                                 |
| `page_load_frontend_time`         | ミリ秒 | ページロード中のフロントエンド領域の平均処理時間                                        | **domInteractive**から**loadEventEnd**までの所要時間                           |
| `page_load_redirect_time`         | ミリ秒 | ブラウザがウェブページを読み込むときに、リダイレクトプロセスに所要した平均時間                         | **redirectStart**から**redirectEnd**までの所要時間                             |
| `page_load_cache_time`            | ミリ秒 | ブラウザがウェブページを読み込む際に、キャッシュされたリソースを検索するのに所要する平均時間                  | **fetchStart**から**domainLookupStart**までの所要時間                          |
| `page_load_dns_time`              | ミリ秒 | ブラウザがウェブページを読み込む際に、ウェブサイトドメインを照会するのに所要する平均時間                    | **domainLookupStart**から**domainLookupEnd**までの所要時間                     |
| `page_load_connect_time`          | ミリ秒 | ブラウザがサーバーとのネットワーク接続プロセスに所要する平均時間                                | **connectStart**から**connectEnd**までの所要時間                               |
| `page_load_ssl_time`              | ミリ秒 | ブラウザがウェブページを読み込む際に、TCPヘンドシェイクの過程で所要する平均時間                       | **secureConnectionStart**から**connectEnd**までの所要時間                      |
| `page_load_firstbyte_time`        | ミリ秒 | ブラウザがウェブページを読み込み、ネットワークリクエストを送信した後、サーバーから最初のバイトが受信されるまで所要する平均時間 | **requestStart**から**responseStart**までの所要時間                            |
| `page_load_download_time`         | ミリ秒 | ブラウザがウェブページを読み込む際に、サーバーからリソースをダウンロードするのに所要する平均時間                | **responseStart**から**responseEnd**までの所要時間                             |
| `page_load_domcontentloaded_time` | ミリ秒 | ウェブページが初期コンテンツをレンダリングして、相互作用できる状態になるまでの平均時間                     | **domContentLoadedEventStart**から**domContentLoadedEventEnd**まで所要された時間 |
| `page_load_domload_time`          | ミリ秒 | ウェブページのすべてのリソースがロードされ、レンダリングが完了した後、ユーザーが相互作用できる状態になるまでの平均時間     | **loadEventStart**から**loadEventEnd**までの所要時間                           |
| `page_load_render_time`           | ミリ秒 | サーバーからダウンロードしたリソースを画面にレンダリングし、ページロードイベントを完了するのに所要する平均時間         | **domInteractive**から**loadEventEnd**までの所要時間                           |
| `page_load_event_time`            | ミリ秒 | ブラウザが測定するloadイベントが完了する平均時間                                      | 開始から**loadEventEnd**までの所要時間                                           |
| `page_load_duration`              | ミリ秒 | ブラウザがウェブページを完全に読み込むのに所要する平均時間                                   | **loadEventEnd**以降、追加のインタラクションを完了するまでの時間、ブラウザエージェントが測定したページロード完了時間    |
| `page_load_count`                 | 件数  | ページロード回数                                                        | -                                                                     |

</div>

## rum_router_change_each_page

-   収集間隔：5秒

-   統計データ：5分、1時間

### Tags

| タグ名                 | 説明        | 備考                                      |
| ------------------- | --------- | --------------------------------------- |
| `page_group`        | ページグループ   | -                                       |
| `completed_loading` | 切り替え完了の有無 | ページ切り替えが完了せずに別のページに切り替えると、`false`と表示される |

### Fields

<div class='browser-metric'>

| フィールド名                | 単位  | 説明                     | 備考 |
| --------------------- | --- | ---------------------- | -- |
| `router_change_count` | 件数  | SPAページの切り替え数           | -  |

</div>

## rum_resource_each_page

ウェブページで呼び出されるリソースの応答時間に関する指標です。

-   収集間隔：5秒

-   統計データ：5分、1時間

### Tags

| タグ名            | 説明                     | 備考                                             |
| -------------- | ---------------------- | ---------------------------------------------- |
| `page_group`   | ページグループ                | -                                              |
| `is3rdParty`   | サードパーティから呼び出されたリソースの有無 | ウェブページのドメインと呼び出されたリソースのドメインが異なる場合は、'true'で配信する |
| `request_host` | リソースのurl host値         | -                                              |
| `request_path` | リソースのurl path値         | -                                              |
| `type`         | リソースタイプ                | css, image, script等                            |

### Fields

<div class='browser-metric'>

| フィールド名                   | 単位  | 説明                                                              | 備考                                                               |
| ------------------------ | --- | --------------------------------------------------------------- | ---------------------------------------------------------------- |
| `resource_dns_time`      | ミリ秒 | ブラウザがウェブページを読み込む際に、ウェブサイトドメインを照会するのに所要する平均時間                    | **domainLookupStart**から**domainLookupEnd**までの所要時間                |
| `resource_connect_time`  | ミリ秒 | ブラウザがサーバーとのネットワーク接続プロセスに所要する平均時間                                | **connectStart**から**connectEnd**までの所要時間                          |
| `resource_ttfb_time`     | ミリ秒 | ブラウザがウェブページを読み込み、ネットワークリクエストを送信した後、サーバーから最初のバイトが受信されるまで所要する平均時間 | **requestStart**から**responseStart**までの所要時間                       |
| `resource_download_time` | ミリ秒 | ブラウザがウェブページを読み込む際に、サーバーからリソースをダウンロードするのに所要する平均時間                | **responseStart**から**responseEnd**までの所要時間                        |
| `resource_duration`      | ミリ秒 | ブラウザがウェブページを完全に読み込むのに所要する平均時間                                   | _loadEventEnd_以降、追加のインタラクションを完了するまでの時間、ブラウザエージェントが測定したページロード完了時間 |
| `page_load_count`        | 件数  | ページロード回数                                                        | -                                                                |
| `resource_size`          | バイト | リソースサイズ                                                         | -                                                                |

</div>
