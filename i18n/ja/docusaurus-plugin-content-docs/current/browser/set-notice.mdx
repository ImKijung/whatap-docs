description: イベント条件を設定し、さまざまな形式で通知を受け取ることができます。
tags:
  - ブラウザーモニタリング
  - 通知
---

-   **_イベント設定_**メニューにより、しきい値ベースの通知条件および受信設定を行うことができます。
-   **_イベント受信設定_**メニューでは、個人通知の受信チャネルおよびタイムゾーンを設定できます。
-   **_イベント履歴_**メニューでは、通知発生履歴を確認できます。

:::note

-   다른 계정에 대한 알림을 설정하려면 **수정**, **알림 설정**, **멤버 관리** 권한 중 하나가 있어야 합니다.
-   イベントを追加または設定するには、通知設定の権限が必要です。 権限体系の詳細については、[次の文書](../project/project-structure)を参考にしてください。

:::
