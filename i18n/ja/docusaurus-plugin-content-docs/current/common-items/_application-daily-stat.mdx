アプリケーションの主な性能指標の1日の推移を時間単位チャートで確認することができます。

<ImgLang img='application_daily_app_stat.png' desc='日別APP現況' />

一日間のアプリケーションの状態を確認できます。 検索条件(<Cmdname sid="select_date2" className="uitext" />, <Cmdname sid="option_type" className="uitext" />, <Cmdname sid="application" className="uitext" />)を設定し![検索アイコン](/img/ico-btn-search.png)ボタンを選択してください。 <Cmdname sid="option_type" className="uitext" />オプションの条件は次のとおりです。

| 分類                                              | 説明                                              |
| ----------------------------------------------- | ----------------------------------------------- |
| <Cmdname sid="type_oid" className="uitext" />   | プロジェクトに含まれているアプリケーションエージェント名                    |
| <Cmdname sid="type_okind" className="uitext" /> | エージェント設定で`whatap.okind`オプションで分類されたグループ単位        |
| <Cmdname sid="type_onode" className="uitext" /> | エージェント設定で`whatap.onode`オプションで分類されたグループ単位        |

提供するチャートは次のとおりです。 チャートの右側に位置する数値は、一日の間に累積された値または平均、最大値を確認することができます。
