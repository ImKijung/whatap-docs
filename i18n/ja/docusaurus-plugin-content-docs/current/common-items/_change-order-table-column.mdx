1. テーブル右上の ![設定アイコン](/img/ico-setting.svg)ボタンを選択してください。

2. ドロップダウンメニューから<Cmdname sid="column_settings" className="uitext" />を選択してください。

3. <Cmdname sid="column_settings" className="uitext" /> ウィンドウが表示されたら <Cmdname sid="column_order_setting" className="uitext" /> セクションでお探しの項目をドラッグして順序を変更してください。

4. すべての設定を完了して ![閉じるアイコン](/img/ico-close.svg) ボタンを選択してください。

設定した内容がテーブルに適用されます。
