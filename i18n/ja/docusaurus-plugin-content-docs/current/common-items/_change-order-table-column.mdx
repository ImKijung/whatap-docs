<Title hashid='change-order-column' level={props.level}>テーブルカラムの順序付け</Title>

<InDoc product="browser">

ブラウザーエラー一覧のテーブルでカラム ヘッダーの順序を変更できます。

</InDoc>

1. テーブル右上の ![設定アイコン](/img/ico-setting.svg)ボタンを選択してください。

2. ドロップダウンメニューから<Cmdname sid="column_settings" className="uitext" />を選択します。

3. <Cmdname sid="column_settings" className="uitext" /> 画面が表示されたら <Cmdname sid="column_order_setting" className="uitext" /> セクションでお探しの項目をドラッグして順序を変更します。

4. すべての設定を完了して![閉じるアイコン](/img/ico-close.svg)ボタンを選択します。

設定した内容がテーブルに適用されます。
