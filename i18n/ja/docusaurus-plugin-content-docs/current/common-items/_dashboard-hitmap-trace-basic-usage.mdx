-   ![数字1](/img/number-01.png)**トランザクション照会件数**:ヒットマップウィジェットからドラッグした領域のトランザクション一覧の数を確認できます。 最大1000件まで照会できます。

-   ![数字2](/img/number-02.png)**トランザクション分類**:![アクティブスタックアイコン](/img/ic-s-active-stack.svg)**_アクティブスタック_**と![マルチトランザクションアイコン](/img/ic-s-mult-transation.svg)**_マルチトランザクション_**を区分して一覧を確認することができます。

-   ![数字3](/img/number-03.png)**トランザクション検索**:トランザクション名、エージェント名、エラー メッセージに基づいてトランザクション一覧を検索できます。

-   ![数字4](/img/number-04.png)**一覧ソート**:テーブルのヘッダーカラム項目をクリックすると、選択した項目に基づいてトランザクション一覧を並べ替えることができます。

-   エラーが発生したトランザクション項目は、一覧で赤色で表示されます。

    <ImgLang img="analysis-tx-main-error.png" desc="トランザクション一覧" />

:::tip

-   テーブルで頻繁に見たいカラム項目を前方に配置したり、追加、非表示にしたりできます。 詳細は[次の文書]](#set-table-column)を参照してください。

-   **_ヒットマップ_**ウィジェットから別の領域をドラッグすると、新しいトレース分析ウィンドウを読み込むことができます。 既存のトレース分析ウィンドウと比較して確認できます。

:::
