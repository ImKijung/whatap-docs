### メソッド要約 {#anlaysis-trace-method-summary}

<ImgLang img="analysis-tx-method-summary.png" desc="メソッド要約" />

メソッド情報のみを確認するには、<Cmdname sid="TTL06130" className="uitext" />タブを選択してください。 エージェントに追跡が設定されたメソッド名と所要時間を表示します。 不要なロジックが繰り返し実行される場合、低い件数に比べて実行時間が長くかかる場合を把握し、メソッドロジック改善のための分析情報として活用できます。
