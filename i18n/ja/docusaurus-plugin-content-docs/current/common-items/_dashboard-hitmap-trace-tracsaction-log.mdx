### TXログ

トランザクションに関するログ情報を確認するには、**_トランザクションログ_**タブを選択してください。

<ImgLang img="analysis-tx-transaction-log.png" desc="トランザクションログ" />

-   **キーワード検索**:キーワード検索欄に検索したいテキストを入力し、エンターキーを入力したり、![検索アイコン](/img/ico-search.svg)ボタンを選択してください。 入力したテキストと一致するキーワードをハイライト表示します。

-   ![ハイライトアイコン](/img/ic-highlight.svg)**_キーワード·ハイライト_**:キーワードと色を設定すると、自動的にログ一覧でキーワードと一致するテキストをハイライト表示します。

-   ![設定アイコン](/img/ico-setting.svg)**_テーブル設定_**:ログ一覧テーブルのヘッダーカラムを設定できます。

:::note

-   **_トランザクションログ_**タブを有効にするには、ログに関連するエージェント設定オプションを適用する必要があります。 ログ設定の詳細については、[次の文書](../log/log-function)を参照してください。

-   **_トランザクションログ_**タブは**\*ログ照会**権限を持つメンバーのみ進入できます。 メンバー権限の詳細については、[次の文書](../project/project-structure#member-auth)を参照してください。

:::
