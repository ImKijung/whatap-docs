#### キーワードハイライト設定する{#anlaysis-trace-set-keyword}

1.  ハイライト表示するキーワードを追加するには![ハイライトアイコン](/img/ic-highlight.svg)ボタンを選択してください。

    <ImgLang img="analysis-tx-transaction-log-highlight-keyword.png" desc="キーワード·ハイライト" />

2.  ご希望のキーワードを入力してください。

3.  色アイコンを選択し、ご希望の色を選択してください。

4.  エンターを入力してください。

追加したキーワードが一覧に作成されます。

#### キーワードハイライト削除する{#anlaysis-trace-delete-keyword}

ハイライト一覧から利用しない項目を削除するには、![削除アイコン](/img/ic-minus.svg)ボタンを選択してください。
