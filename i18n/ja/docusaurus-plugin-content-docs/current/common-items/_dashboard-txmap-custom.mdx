### ユーザーカスタム項目のモニタリング{#custom}

特定のトランザクション、ドメイン、IPを<Cmdname sid="custom" className="uitext" />タブに追加して、応答時間の分布図チャートで選択した項目だけを確認できる機能を提供します。 これにより、ユーザーは問題が発生する特定のアイテムを効率的にモニタリングして分析できます。

1. <Cmdname sid="custom" className="uitext" />タブを選択します。

2. <Cmdname sid="transaction" className="uitext" />または<Cmdname sid="domain" className="uitext" />, <Cmdname sid="ip" className="uitext" />項目で希望する値を直接入力するか<UI>Top 30</UI>一覧から提供する項目を選択できます。

   <ImgLang img="txmap-custom-filter-add.png" desc="カスタム" />

   :::note

   希望する項目を検索して選択できます。

   :::

3. 追加した項目を選択します。 

   <ImgLang img="txmap-custom-filter-select.png" desc="カスタム" />

選択した項目の応答時間の分布図が、右のチャートにリアルタイムでフィルタリングされることを確認できます。

:::note

追加したカスタム項目を削除するには、![削除アイコン](/img/ico-trash.svg)ボタンを選択しまます。

:::
