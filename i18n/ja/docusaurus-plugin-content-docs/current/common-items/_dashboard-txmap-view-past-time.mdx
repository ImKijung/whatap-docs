## 過去のデータ参照

<Cmdname sid="transaction_map" className="uitext" />は、リアルタイムモニタリングを基本的に提供します。 既定値として5分間のデータをリアルタイムで照会できます。 タイムセレクターの緑のボタンをクリックして、希望の照会時間を選択してください。

過去のデータを照会するには、タイムセレクターで![一時停止アイコン](/img/ico-pause.svg)ボタンを選択してください。 非リアルタイムモードに変更され、最大10分までのトランザクションデータを照会できます。

<ImgLang img='txmap-past-time-selector.png' desc='過去のデータ参照' />

希望する日時を設定した上で<Cmdname sid="apply" className="uitext" />ボタンを選択してください。 ユーザーが設定した時間を基準に<Cmdname sid="transaction_map" className="uitext" />のチャートデータを更新します。

:::note

- 過去の照会範囲は、トランザクションの量によって異なる場合があります。

- 過去のデータを照会してからリアルタイムモードに切り替えるには、![再生アイコン](/img/ico-play-time.svg) ボタンを選択してください。

:::
