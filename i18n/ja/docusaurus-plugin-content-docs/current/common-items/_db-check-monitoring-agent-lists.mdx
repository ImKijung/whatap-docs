## モニタリングの確認

サーバーで正常にログが確認され、コンソールに正しく登録されたことを確認するには、[WhaTapモニタリングサービス](https://service.whatap.io)の初期画面に移動します。確認するプロジェクトを選択し、画面の左側の<Cmdname sid="side_management" className="uitext" /> > <Cmdname sid="side_dbAgents" className="uitext" />メニューを選択します。<Cmdname sid="side_dbAgents" className="uitext" />からデータベースサーバーの名前を確認できます。データベースサーバーの名前を別途設定する場合、[次の文書](agent-dbx-settings#naming)を参照してください。
