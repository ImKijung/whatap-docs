## インスタンス性能指標の比較

2つ以上のインスタンスを選択して、性能指標を比較できます。性能指標を比較するインスタンスを一覧から選択すると、画面右上の<Cmdname sid="multi_view" className="uitext" />ボタンが活性化されます。<Cmdname sid="multi_view" className="uitext" />ボタンを選択すると、 <Cmdname sid="side_multiInstance" className="uitext" />メニューに移動します。主要な性能指標のデータを比較したグラフチャートとアクティブセッション、ロックツリーに関する詳細情報を確認できます。

:::note

<Cmdname sid="side_multiInstance" className="uitext" />メニューの詳細については、<Link to="multi-instance-monitoring">次の文書</Link>を参考にしてください。

:::
