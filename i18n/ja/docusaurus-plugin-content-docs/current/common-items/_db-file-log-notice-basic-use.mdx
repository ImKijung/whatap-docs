## 基本画面ガイド

<Xclude product='postgresql'>

ホーム画面 > プロジェクト選択 > **_アラート通知_** > **_ファイルログ通知_**

</Xclude>

<InDoc product='postgresql'>

ホーム画面 > プロジェクト選択 > **_サイトマップ_** > **_アラート通知_** > **_ファイルログ通知_**

</InDoc>

![ファイルログ通知](/img/db-file-log-notice.png)

-   **_時間_**：参照時間を選択することができます。

-   **_インスタンス_**：表示するインスタンスを選択してください。

-   **_ログファイル一覧_**：インスタンスのログファイルの一覧を確認することができます。

-   **_ログ発生履歴_**：設定したファイル名と設定したキーワードでログが発生した時間を確認することができます。
