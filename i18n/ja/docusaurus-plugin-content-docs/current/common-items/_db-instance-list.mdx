ホーム画面 > プロジェクト選択 > **_ダッシュボード_** > **_インスタンス一覧_**

![インスタンス一覧](/img/db-dashboard-instance-list.png)

エージェントをインストールしたデータベースサーバーの一覧を確認できます。主なパフォーマンス指標とステータスを確認できます。パフォーマンス指標は5秒間隔で更新され、一覧のカラムは設定を使用して順序を変更したり、任意の指標に変更することができます。

:::note

**_状態_**カラムで正常でない状態アイコンをクリックすると**_イベント_**ウィンドウが表示されます。当該インスタンスで発生したイベントの履歴を確認できます。

:::
