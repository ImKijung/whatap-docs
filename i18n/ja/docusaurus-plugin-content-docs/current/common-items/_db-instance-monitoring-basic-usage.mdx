-   ![数字3](/img/number-03.png)**_アクティブセッション_**|**_ロックツリー_**|**_プロセス情報_**

    リアルタイムのセッションやロックツリー、プロセス情報を表示することができます。

    -   **_アクティブセッション_**

        リアルタイムで実行中のセッションとクエリ内容を表示することができます。テーブルの右側の**_query_**カラムの項目を選択すると、**_SQLビュー_**ウィンドウが表示されます。選択したクエリ(Query)のFull TextおよびPlan情報を確認できます。テーブルのカラムの詳細については、[次の文書](#column-info)を参照してください。

    -   **_ロックツリー_**

        ロックセッションに関する情報とHolder、Waiter情報をリアルタイムで表示することができます。テーブルのカラムの詳細については、[次の文書](#column-info)を参照してください。

    -   **_プロセス情報_**

        データベースサーバーのリソース使用量をリアルタイムで表示できます。
