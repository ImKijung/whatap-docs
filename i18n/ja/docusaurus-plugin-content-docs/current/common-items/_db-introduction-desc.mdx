データベースのパフォーマンス管理は、データベースのパフォーマンスをリアルタイムでモニタリングし、問題点を診断、分析します。問題のあるSQLをチューニングして適用し、問題が解決されたかをリアルタイムでモニタリングします。 

WhaTapデータベースモニタリングサービスは、これら一連のタスクを循環管理できるように設計されています。データベースの動作をモニタリング、診断、分析し、可用性とパフォーマンスを一定のレベルに維持できるように構成されています。データベースに問題が発生した場合、担当者にアラームを送信して状況を認識し、収集されたデータからデータベースの状況をよりに分析することで、Root Causeを明確にする支援サービスです。
