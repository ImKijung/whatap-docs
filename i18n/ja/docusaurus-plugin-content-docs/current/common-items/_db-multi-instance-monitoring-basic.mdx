## 基本画面ガイド

<ProdImg img='db-multi-instance.png' desc='マルチインスタンス' />

- ![数字1](/img/number-01.png)**_インスタンス_**

  エージェントをインストールしたインスタンス(データベース) の一覧です。インスタンスの色は、画面に配置されているグラフチャートの色と一致します。グラフチャートから一部のインスタンスのデータを除外する場合は、そのインスタンスを選択してください。インスタンスを選択すると、グラフチャートにインスタンスのデータを反映できます。

- ![数字2](/img/number-02.png)**_ステータス_**

  データベースサーバーのリソースに関する指標をモニタリングできます。

- ![数字3](/img/number-03.png)**_ロードバランス_**

  どのインスタンスの指標値が高いかを確認し、1台のサーバーに負荷が集中していないかモニタリングできます。

- ![数字4](/img/number-04.png)グラフチャートセクション

  各ウィジェットで横軸は時間、縦軸は各指標の数値を意味します。 

<Xclude product='redis'>

- ![数字5](/img/number-05.png)**_アクティブセッション_**|**_ロックツリー_**|**_プロセス情報_**

  特定のインスタンスでリアルタイムで実行中のセッション、ロックツリー、プロセス情報を参照できます。「インスタンス名」ボタンを選択してください。

</Xclude>

<InDoc product='redis'>

- ![数字5](/img/number-05.png)**_アクティブセッション_**

  特定のインスタンスでリアルタイムのセッション情報を表示できます。「インスタンス名」ボタンを選択します。

</InDoc>

:::note

- 各メトリクスの詳細については、[次の文書](metrics-data-list)を参照してください。

- カラム項目の詳細については、[次の文書](#column-info)を参照してください。

- **_プロセス情報_**は、XOSエージェントを追加して設定しないと確認できません。詳細については、[次の文書](agent-xos-settings)を参照してください。
