- **pg_object** <span class='type'>Boolean</span>

  既定値`false`

  Topオブジェクト情報の収集有無を設定します。 情報を収集するには`true`に変更してください。

- **pg_object_hour** <span class='type'>Int</span>

  既定値`5`

  Topオブジェクト情報の収集時間を設定します。 既定値は`5`で、午前5時に収集を始めます。

- **table_bloat_ratio** <span class='type'>Percentage</span>

  既定値`50`

  テーブルbloating ratio値が設定値以上の場合は、情報を収集します。 既定値は50%です。

- **table_bloat_bytes** <span class='type'>Byte</span>

  既定値`10485760`

  テーブルbloatingbytes値が設定値以上の場合、情報を収集します。 既定値は10MBです。

- **index_bloat_ratio** <span class='type'>Percentage</span>

  既定値`50`

  インデックスbloatingratio値が設定値以上の場合、情報を収集します。 既定値は50%です。

- **index_bloat_bytes** <span class='type'>Byte</span>

  既定値`10485760`

  インデックスbloating bytes値が設定値以上の場合、情報を収集します。 既定値は10MBです。

- **autovacuum_list_limit**  <span class='type'>Int</span>

  既定値`50`

  日付の古いage(`relfrozenxid`)順に設定値程のテーブル情報を収集します。 既定値は50件です。
