-   **大容量規模でも効率的な構造**

    複数のDBサーバーを管理する場合でも、一元化された性能データ収集および効率的なデータストレージ構造により、最適な分析環境を提供します。

-   **クラウドDBモニタリング可能**

    Queryベースのモニタリングデータ収集構造では、DBサーバーの種類に制限はありません。AWS、GCP、Azureおよび、OCI環境のDBをモニタリングでき、クラウドリソース指標と統合モニタリングにも対応しています。

-   **統合モニタリング**

    アプリケーション、サーバー、データベースモニタリング製品をすべてインストールすることで、サービスの異常を特定する際に接続されたビューを使用して障害の原因を迅速に分析できます。

-   **WhaTap SaaS**

    さまざまなデータベース製品を選択でき、毎月のサブスクリプションの利用期間を自由に設定できます。(インストールタイプも提供可能することができます)
