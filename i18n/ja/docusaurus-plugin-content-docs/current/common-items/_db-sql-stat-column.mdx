| 項目                | 説明                                                 |
| ----------------- | -------------------------------------------------- |
| execute count     | SQLを実行した回数                                         |
| elapse avg (sec)  | SQLを実行した平均実行時間(秒)で`elapse sum / execute count`の結果値 |
| elapse sum (sec)  | SQLを実行した時間(秒) の合計                                  |
| elapse max (sec)  | SQLの最大実行時間(秒)                                      |
| elapse wait (sec) | SQLの待機時間(秒)                                        |
| instance          | インスタンス名                                            |
