## 基本画面ガイド

<ProdImg img='db-database-sql-stat.png' desc='SQL統計' />

<Xclude product='postgresql,mysql'>

1.  **_インスタンス_**一覧から表示する対象を選択してください。重複するインスタンスを選択することができ、すべてのインスタンスを選択するには**_全体_**を選択してください。

2.  **_開始日_**、**_開始時間_**、**_期間_**で表示日付と時間、期間を設定してください。

3.  ![検索アイコン](/img/ico-btn-search.png)ボタンを選択してください。

</Xclude>

<InDoc product='postgresql,mysql'>

1.  ![数字1](/img/number-01.png) **_時間_**で、データを表示する時間と日付を選択してください。緑色のボタンを選択して検索時間を選択することもできます。

2.  ![数字1](/img/number-01.png) **_インスタンス_**でデータを検索する対象を選択してください。複数のインスタンスを選択でき、すべてのインスタンスを選択するには、**_全体選択_**をクリックしてください。

3.  ![検索アイコン](/img/ico-btn-search.png)ボタンを選択してください。

</InDoc>

上位50件までの内容をテーブルに表示します。

:::note

<Xclude product='postgresql,mysql'>

-   SQLのパラメータ値を**_フィルター_**テキストボックスに入力し、必要な内容をフィルタリングすることができます。

-   デフォルトのソート順は実行回数です。ソート順を変更するには、 **_ソート順 _**の一覧から目的の条件を選択してください。

-   参照した内容をcsv形式のファイルにダウンロードするには、![csvダウンロードアイコン](/img/ic-csv-download-ms.svg)ボタンを選択してください。

</Xclude>

<InDoc product='postgresql,mysql'>

-   任意の日付と時間を表示するには、時計の表示を選択してください。日付と時間のテキスト領域をクリックすると、日付と時間を選択するオプションが表示されます。

-   テーブルヘッダーの各カラムをクリックすると、選択したカラムをソートすることができます。

-   **_参照基準_**で選択した項目に従ってソートすることができます。ご希望の項目を選択した後![検索アイコン](/img/ico-btn-search.png)ボタンを選択してください。

</InDoc>

:::
