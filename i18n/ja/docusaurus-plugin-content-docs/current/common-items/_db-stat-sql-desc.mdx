データベースサーバーを利用すると、サーバーの容量を多く使うDBに使用量と余裕量を確認する必要があります。定期的にデータをバックアップし、空き容量を確保しないと、サービスが中断される可能性があります。提供する次の機能により、データベースの使用量を定期的に確認および管理してください。

-   SQL別の時系列チャートで発生時間および性能を確認できます。 

-   複数のインスタンスの統計情報を合計して表示することができます。

-   使用カテゴリ別にデータを分類および表示することができます。

:::note

5秒ごとに収集されるセッションの情報と実行中のSQL Textを提供する統計ページです。このページは、収集方法によって誤差が生じる可能性があります。

:::
