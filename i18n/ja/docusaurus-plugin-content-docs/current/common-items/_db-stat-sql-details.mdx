チャートの棒グラフを選択すると、5分間の統計情報が確認できる<Cmdname sid="session_detail" className="uitext" />ウィンドウが表示されます。 選択した時間のSQL実行一覧を表示し、項目を選択すると新しいウィンドウとともに<Cmdname sid="instance_monitoring" className="uitext" />メニューに移動します。

<Cmdname sid="instance_monitoring" className="uitext" />メニューでは選択した時間帯のアクティブセッションテーブルと様々な指標の推移を確認することができます。
