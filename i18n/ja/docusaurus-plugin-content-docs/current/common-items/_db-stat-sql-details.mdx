チャートの棒グラフを選択すると、5分間の統計情報が確認できる<Cmdname sid="session_detail" className="uitext" />画面が表示されます。 選択した時間のSQL実行一覧を表示し、項目を選択すると新しい画面とともに<Cmdname sid="instance_monitoring" className="uitext" />メニューへ移動します。

<Cmdname sid="instance_monitoring" className="uitext" />メニューからは、選択した時間帯のアクティブセッションテーブルと、様々な指標の推移を確認することができます。
