## クエリ別の実行推移を確認

<Xclude product='postgresql'>

テーブル各項目別の一番左の![数字1](/img/number-01.png)►ボタンを選択してください。該当クエリ文の5分当たりの実行回数と平均実行時間を確認できるチャートが出ます。各棒グラフにマウスをオーバーすると、その時間の数値を確認できます。

使用区分別にデータを分類して照会するには![数字2](/img/number-02.png)で希望するタブを選択してください。

</Xclude>

<InDoc product='postgresql'>

テーブル各項目別の一番左の![数字3](/img/number-03.png)►ボタンを選択してください。該当クエリ文の5分当たりの実行回数と平均実行時間を確認できるチャートが出ます。各棒グラフにマウスをオーバーすると、その時間の数値を確認できます。

:::note

グラフチャートで希望する時間の棒グラフを選択すると、**_インスタンスモニタリング_**メニューに移動します。選択した時間の指標を確認できます。インスタンスモニタリングの詳細については、[次の文書](instance-monitoring)を参考にしてください。

:::
