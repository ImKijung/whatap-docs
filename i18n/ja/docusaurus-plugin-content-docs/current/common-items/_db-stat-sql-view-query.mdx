## クエリ別の実行推移を確認

<Xclude product='postgresql,mysql'>

テーブルの各項目別の左の![数字1](/img/number-01.png)►ボタンを選択してください。クエリ文の5分当たりの実行回数と平均実行時間が表示されます。各棒グラフにカーソルを合わせると、その時間の数値が表示されます。

使用カテゴリ別にデータを分類して表示するには![数字2](/img/number-02.png)で目的のタブを選択してください。

</Xclude>

<InDoc product='postgresql,mysql'>

テーブルの各項目別の左の![数字3](/img/number-03.png)►ボタンを選択してください。クエリ文の5分当たりの実行回数と平均実行時間が表示されます。各棒グラフにカーソルを合わせると、その時間の数値が表示されます。

:::note

グラフチャートで目的の時間の棒グラフを選択すると、**_インスタンスモニタリング_**メニューに移動します。選択した時間の指標を確認できます。インスタンスモニタリングの詳細については、[次の文書](instance-monitoring)を参照してください。

:::

使用カテゴリ別にデータを分類して表示するには![数字2](/img/number-02.png)で目的のタブを選択してください。

</InDoc>
