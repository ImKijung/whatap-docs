WhaTapのデータベースモニタリングサービスは、エージェントと収集サーバーで構成され、エージェントと収集サーバーの間でセキュリティのためにデータを暗号化します。外部ライブラリを使ってないため、インストールおよびアップデートする際にシステムの再起動は要りません。

-   WhaTapデータベースモニタリングサービスは、データベースサーバーにエージェントをインストールしなくても、別のエージェントサーバーでモニタリングすることができます。ユーザーの環境に柔軟に対応できます。

    |                      他のエージェントサーバーにインストール可能                      |                        DBサーバーにインストール可能                       |           DBリソースモニタリング(XOS) 可能          |

    | <ImgLang img='dbx-install-agent-on-agent-server.png' desc='' /> | <ImgLang img='dbx-install-agent-on-db-server.png' desc='' /> | <ImgLang img='xos-agent.png' desc='' /> |

    </div>

-   XOSエージェントは、データベースサーバーのプロセス使用状況をモニタリングできる付加オプションエージェントです。データベースサーバーでプロセス使用状況をモニタリングする場合は、データベースサーバーに別のエージェントを起動してデータを収集できます。

-   エージェントとサーバーは、バイナリベースの通信プロトコルを使用します。また、データを選択的に暗号化することで、データのセキュリティを確保しながら、暗号化の負荷を低減します。

:::note

-   モニタリング対象のデータベースサーバーとネットワーク通信が可能なサーバーにWhaTapエージェントをインストールする必要があります。

-   WhaTap DBXエージェントサーバには、**Java 8バージョン**以降の**JDK**をインストールする必要があります。

:::
