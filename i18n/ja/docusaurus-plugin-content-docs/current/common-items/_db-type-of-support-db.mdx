## サポートされているデータベースの種類

WhaTapデータベースモニタリングサービス(DBX)では、パフォーマンス分析のために継続的なStat指標とセッションデータを収集し、収集されたデータを保存します。特定期間の全体的な推移を分析または比較することができます。また、特定の時点で進行中だったセッションを特定し、遅いQueryを分析することができます。収集されたSQLを再加工してSQLの統計データを分析することが可能です。

DBXは、次のデータベースをサポートします。**Java8**以降の**JDK**をインストールする必要があります。
