### 比較対象の選択

![数字1](/img/number-01.png) **_基準日_**、**_インスタンス_**を選択し、![数字2](/img/number-02.png) **_比較日_**、**_インスタンス_**を選択してください。 グラフにカーソルを合わせると、表示されるツールチップでデータを比較できます。

<ImgLang img='db-compare-count-trend-tootip.png' dsec='Tooltip' />

-   グラフチャートの実線は**_基準日_**のデータ、点線は**_比較日_**のデータを表示します。

-   ツールチップに**_(Comparison)_**が表示されている項目は、**_比較日_**のデータです。

:::note

-   **_データベース_**項目で比較したいデータベースを選択することもできます。

-   **_データベース_**項目の各色は、グラフチャートで同じ色で表現されます。

:::
