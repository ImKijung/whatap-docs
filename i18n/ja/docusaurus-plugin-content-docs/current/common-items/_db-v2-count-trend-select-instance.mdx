### 比較対象の選択

![数字1](/img/number-01.png) **_基準日付_**、**_インスタンス_**を選択した上で![数字2](/img/number-02.png) **_比較日付_**、**_インスタンス_**を選択してください。 チャートの上にマウスオーバーすると表示されるツールチップを通じてデータを比較できます。

<ImgLang img='db-compare-count-trend-tootip.png' dsec='Tooltip' />

-   グラフチャートの実線は**_基準日付_**のデータ、点線は**_比較日付_**のデータを表示します。

-   ツールチップで**_(Comparison)_**が表示された項目が**_比較日付_**のデータです。

:::note

-   **_データベース_**項目で比較したいデータベースを選択することもできます。

-   **_データベース_**項目の各色は、グラフチャートで同じ色で表現されます。

:::
