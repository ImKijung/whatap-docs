**_[インスタンス](instance-monitoring)_**メニューは、1つのインスタンスだけをモニタリングすることに対し、**_マルチインスタンス_**は、複数のインスタンスの指標を1つの画面で確認して比較することができます。 動作中のエージェント数、アクティブセッションの合計、トランザクション情報などのインスタンス指標をグループ化した値を通じて、インスタンスの現況を把握することができます。 グラフチャートを通じて、インスタンスの指標値の推移はもちろん、どのインスタンスの指標値が高いかを簡単に確認できます。 最も長く持続するアクティブセッション情報を提供します。
