### セッション詳細情報を確認する{#session-details}

<Cmdname sid="active_sessions" className="uitext" />タブでセッション項目を選択すると<Cmdname sid="session_detail" className="uitext" />ウィンドウが表示されます。 選択したセッションについての詳細を確認できます。

<ImgLang img='db-session-details-info.png' desc='セッション詳細' />

- セッションに関する情報は、既定値でリアルタイムで照会されています。 タイムセレクターの![静止アイコン](/img/ico-pause.svg)ボタンを選択すると、リアルタイム照会を中止し、中止した時間ベースのデータを確認することができます。 リアルタイム照会を中止した状態でデータを更新するには、![更新アイコン](/img/ico-refresh.svg)ボタンを選択してください。

- **_Kill Session_**：選択したセッションを停止できます。 <Cmdname sid="session_kill" className="uitext" />ウィンドウが表示されたらパスワード(**_Parameter Key_**) 値を入力した上で<Cmdname sid="apply" className="uitext" />ボタンを選択してください。

  :::note

  この機能は、リアルタイム照会モードでのみ利用できます。

  :::

<InDoc product="postgresql">

- <Cmdname sid="TTL06721" className="uitext" />：選択したセッション履歴が確認できる<Cmdname sid="side_sessionHistory" className="uitext" />メニューに移動できます。

  :::note

  <Cmdname sid="side_sessionHistory" className="uitext" />メニューの詳細については、<Link to="analysis-session-history">次の文書</Link>を参照してください。

  :::

</InDoc>

- **_Session Info_**：選択したセッションについての詳細を確認できます。

- **_Wait Info_**：当該バックエンドプロセスが現在待機中の場合、待機イベント名(`wait_event`)とイベントタイプ(`wait_event_type`)情報を確認することができます。

- **_Process_Info_**：選択したセッションのプロセス情報を確認できます。

- **_SQL Text_** / **_Plan_**：選択したセッションのSQLクエリ文とPlan情報を確認することができます。 詳細については[次の文書](#query-details)を参照してください。
