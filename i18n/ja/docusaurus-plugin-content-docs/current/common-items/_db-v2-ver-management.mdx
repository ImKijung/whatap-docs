## V2以前のサービス画面を利用する

ホーム画面 > プロジェクト選択 > **_管理_** > **_プロジェクト管理_**

<ImgLang img='db-v2-ver-management.png' desc='プロジェクトバージョン管理' />

以前のバージョンとの互換性のため、データベースモニタリングV2以前のサービス画面を利用することができます。 

1.  **_管理_** > **_プロジェクト管理_**メニューに移動してください。

2.  **_プロジェクトバージョン管理_**セクションで**_プロジェクトバージョン_**を**_V1_**に選択してください。 

3.  **_保存_**ボタンを選択します。

:::note

-   過去のデータを表示した時に画面に表示されないデータが見つかった場合は、**_プロジェクトのバージョン_**を**_V1_**に変更してから、以前のバージョンの画面でデータを表示できます。

-   互換性の理由から、この機能は 2023.12.31 までしか利用できません。

:::
