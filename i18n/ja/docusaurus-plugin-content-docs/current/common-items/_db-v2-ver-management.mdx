## V2 이전 서비스 화면 이용하기

ホーム画面 > プロジェクト選択 > **_管理_** > **_プロジェクト管理_**

![プロジェクトバージョン管理](/img/db-v2-ver-management.png)

以前のバージョンとの互換性のため、データベースモニタリングV2以前のサービス画面を利用することができます。 

1.  **_管理_** > **_プロジェクト管理_**メニューに移動してください。

2.  **_프로젝트 버전 관리_** 섹션에서 **_프로젝트 버전_**을 **_V1_**으로 선택하세요. 

3.  **_保存_**ボタンを選択します。

:::note

-   과거 데이터 조회 시 화면에 조회되지 않는 데이터가 있다면 **_프로젝트 버전_**을 **_V1_**으로 변경한 다음 이전 버전의 화면에서 데이터를 조회할 수 있습니다.

-   호환성 제공을 위한 이 기능은 2023.10.30.까지만 제공됩니다.

:::
