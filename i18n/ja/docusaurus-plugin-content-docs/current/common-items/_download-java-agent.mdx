## エージェントダウンロード{#download-agent}

**アクセスキー**を発行すると、**_WhaTapエージェントファイルをダウンロード_**セクションで**_ダウンロード_**ボタンがアクティブになります。

1.  **_ダウンロード_**ボタンを選択して、WhaTapエージェントファイルをダウンロードしてください。

    :::tip

    次のコマンドを使用して、サーバーから直接にダウンロードすることもできます。

    ```bash
    wget https://service.whatap.io/agent/whatap.agent.java.tar.gz
    ```

    :::

2.  モニタリング対象サーバーに接続してください。

3.  目的のモニタリング対象サーバーのパスにエージェントファイルを圧縮して解凍してください。圧縮を解凍したパスは`$WHATAP_HOME`になります。

    ```bash
    tar -zxvf whatap.agent.java.tar.gz
    ```
