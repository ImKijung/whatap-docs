## Flexボードコピー {#copyflexboard}

ユーザーが作成したFlexボードをコピーして、他のメンバーに渡すことができます。

<Cmdname sid="integratedFlexboard" className="uitext" />メニューの<Cmdname sid="dashboard_list" className="uitext" />リストから<LinkImage img="ico-copy.svg" className="ico" desc="コピーアイコン" />ボタンをクリックします。<Cmdname sid="copy_to_me" className="uitext" />または<Cmdname sid="copy_to_others" className="uitext" />オプションを選択します。

- <Cmdname sid="copy_to_me" className="uitext" />オプションを選択すると、<Cmdname sid="dashboard_list" className="uitext" />に'_copied'サフィックスが付いた項目が追加されます。

- <Cmdname sid="copy_to_others" className="uitext" />オプションを選択した場合は、<LinkImage img="ico-add.svg" className="ico" /> <Cmdname sid="account" className="uitext" />をクリックします。

  <ImgLang img='flexboard-copy2others.png' desc='他人にコピー' />

  他のメンバーのメールを選択してから<Cmdname sid="copy_dashboard" className="uitext" />ボタンをクリックしてください。コピーをもらったメンバーの<Cmdname sid="dashboard_list" className="uitext" />リストに'\_copied'の接尾語が付いた項目が追加されます。

## JSONファイルで共有する

Flexボード設定をJSONファイルとして保存し、他のメンバーから渡すか、他のメンバーの設定を読み込みできます。

### 抽出

1. <Cmdname sid="integratedFlexboard" className="uitext" />またはプロジェクトの<Cmdname sid="flexboard" className="uitext" />メニューに移動してください。

2. <Cmdname sid="dashboard_list" className="uitext" />で<LinkImage img="ico-export.svg" desc="내보내기 아이콘" className="ico" />ボタンを選択してください。

3. <Cmdname sid="BTN07940" className="uitext" />または<Cmdname sid="BTN07941" className="uitext" />オプションを選択してください。

   - <Cmdname sid="BTN07940" className="uitext" />: ダッシュボードに設定されたプロジェクト情報を、JSONファイルに含めて共有できます。

   - <Cmdname sid="BTN07941" className="uitext" />: ダッシュボードに設定されたプロジェクト情報を、JSONファイルに含めずに共有できます。

4. JSONファイルがダウンロードされたら、共有する他のメンバーに転送してください。

### インポート

1. <Cmdname sid="integratedFlexboard" className="uitext" />またはプロジェクトの<Cmdname sid="flexboard" className="uitext" />メニューに移動してください。

2. 画面右上の![読み込みアイコン](/img/ico-import.svg)<Cmdname sid="import" className="uitext" />ボタンを選択してください。

3. ダウンロードしたJSONファイルを選択してください。

:::note

この機能は、プロジェクトの**修整**権限を所有するメンバーのみが利用できます。

:::
