提供された事前構成テンプレートを使用します。 初期設定を簡単に実行し、設定を変更して利用することができます。

1. <Cmdname sid="side_flexboard" className="uitext" />メニューをクリックしてください。

2. 左側のテンプレート一覧から目的のテンプレートを選択してください。

   <ImgLang img='flexboard-template.png' desc='Flex ボードテンプレートを選択' />

3. <Cmdname sid="flexboard_management" className="uitext" />ウィンドウが表示されたら、<Cmdname sid="rename" className="uitext" />に必要な名前を入力してください。

   <ImgLang img='flexboard-template-window.png' desc='Flex ボード管理' />

   <Cmdname sid="layout" className="uitext" />項目のウィジェットをマウスで選択し、目的の位置にドラッグして移動することができます。

4. <Cmdname sid="save" className="uitext" />ボタンを選択します。

選択したテンプレート基準のFlexボードを作成します。<Cmdname sid="side_flexboard" className="uitext" />メニューで作成したボード一覧を確認することができます。
