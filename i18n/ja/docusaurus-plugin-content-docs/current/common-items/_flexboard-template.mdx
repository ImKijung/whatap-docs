提供された事前構成テンプレートを使用します。 初期設定を簡単に実行し、設定を変更して利用することができます。

1.  **_Flexボード_**メニューをクリックしてください。

2.  左側のテンプレート一覧から目的のテンプレートを選択してください。

    <ImgLang img='flexboard-template.png' desc='Flex 보드 템플릿 선택' />

3.  **_Flexボード管理_**ウィンドウが表示されたら、**_名前変更_**に希望する名前を入力してください。

    <ImgLang img='flexboard-template-window.png' desc='Flex 보드 관리' />

    -   **_レイアウト_**項目のウィジェットをマウスで選択し、目的の位置にドラッグして移動することができます。
    -   **_オーバーラップ_**トグルボタンを選択してオプションをアクティブにすると、ウィジェットが重なり合うように配置できます。

4.  **_保存_**ボタンを選択します。

選択したテンプレート基準のFlexボードを作成します。**_Flexボード_**メニューで作成したボード一覧を確認することができます。
