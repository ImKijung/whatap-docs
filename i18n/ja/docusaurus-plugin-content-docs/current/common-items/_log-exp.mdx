WhaTapモニタリングサービスの初期画面でプロジェクトを選択して、プロジェクトメニューの下部の{@include:\_exp-path.mdx}メニューを選択してください。 

**_ログトレンド_**メニューでタイプ別に分類されたログの発生件数の推移を通じて、特定のエラータイプの発生パターンを確認したり、時間別に詳細ログデータを確認することができます。 ハイライト機能を使用すると、目的のログをすばやく識別できます。カテゴリ別に収集されたログの推移を参照することができます。 {@include: ../common-items/_log-ui-main-term.mdx}

![ログトレンド sc](/img/log-trend.png) 

## データ照会

-   下部までスクロールすると、次のデータを照会します。
-   ![number-s](/img/number-01.png)で探索するログデータの**_時間_**と収集単位として**_カテゴリ_**を指定します。
-   **_カテゴリ_**を変更すると、選択されたカテゴリ対応するログを参照します。![number-s](/img/number-02.png)バーチャトと![number-s](/img/number-03.png)ログテーブルで確認できます。
-   ![number-s](/img/number-02.png)棒グラフのバーをクリックすると、バーの時間範囲に対応するログを検索します。 
-   ![number-s](/img/number-03.png)ログテーブルの上段左側で照会した総ログ数を確認することができます。
-   ![number-s](/img/number-04.png) サイドメニューからタグでフィルタリングすることでログを確認できます。 検索キーは2つまで選択でき、検索値は複数選択できます。

## ログContentの確認

{@include: ../common-items/_log-ui-log-content.mdx} 

-   ![number-s](/img/number-03.png) ログテーブルの行(ログ)ごとに![アイコン](/img/ic-caret-right.svg)**_もっと見る_**ボタンをクリックします。![アイコン](/img/ic-caret-right.svg)**_詳細_**ボタンをクリックして、![number-s](/img/number-05.png)のように対象ログの全体contentを確認することができます。

## ログをチャートで検索

![ログトレンド棒チャートsc](/img/log-trend-bar.png)

-   棒チャートで![aアイコン](/img/char-a.svg)目的の時間をクリックすると、その時間のログが確認できます。

-   棒チャートの下部のログテーブル上部左側の![bアイコン](/img/char-b.svg)時間選択オプションを使用して、次のように選択した時間帯でさらに細分化してログを検索することができます。

    -   1min：interval（チャート内の棒の間隔）

    -   時間選択オプション：選択された時間範囲を6つの区間に分けた時間帯
