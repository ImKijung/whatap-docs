フィルター入力時に使用できる文法の一覧です。**_ライブテール_**と**_ログ検索_**は同じ文法を使います。

| 文法            | 例                             | 説明                                           |
| ------------- | ----------------------------- | -------------------------------------------- |
| include(+)    | `+ote-gateway`                | `ote-gateway`を含むもののみフィルタリング                  |
| ^             | `@txid:+-6611219762256426545` | `@txid`が`-6611219762256426545`であるもののみフィルタリング |
| exclude(-)    | `-ote-gateway`                | `ote-gateway`を除いたもののみフィルタリング                 |
| startWith(\*) | `ERROR*`                      | `ERROR`で始まるもののみをフィルタリング                      |
| endWith(\*)   | `*ERROR`                      | `ERROR`で終わるもののみをフィルタリング                      |

:::note

include(+)文法使用するとき、'`-`1234'のように負の数を含む文字列を検索するには'`+`'記号を追加して'`+-`1234'で検索してください。

:::
