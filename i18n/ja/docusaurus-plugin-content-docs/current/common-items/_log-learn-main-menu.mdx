WhaTapのログモニタリングは、統合システムの構築によるユーザーの利便性とアクセシビリティを高めました。WhaTapは独自の技術力をベースに確実にデータ収集が行われることで、ユーザーが主に使用するライブテール、ログトレンド、ログ検索、イベント通知、Parserの効率性をサポートします。 

**_ログ_**メニューは、クエリや分析、オプション設定などの機能を提供します。**_イベント設定_**メニューでは、ログ関連のイベント通知を設定することができます。

- ホーム画面 > プロジェクト選択 > **_ログ_**

  WhaTapモニタリングサービスのメイン画面からプロジェクトを選択し、プロジェクトメニューの下にある**_ログ_**メニューを選択してください。次の機能を使用することで複雑なログに簡単にアクセスでき、様々な条件でリアルタイムに確認および分析することができます。

  - **_ライブテール_**
  - **_ログトレンド_**
  - **_ログ検索_**
- ホーム画面 > プロジェクト選択 > **_アラート通知_** > **_イベント設定_**

  WhaTapモニタリングサービスのメイン画面でプロジェクトを選択し、プロジェクトメニューの下にある**_アラート通知_**メニューを選択し、**_イベント設定_**メニューに移動します。 イベント条件を設定し、メール、SMS、メッセンジャー、AppPushなどの様々なパスから通知を受信することができます。
