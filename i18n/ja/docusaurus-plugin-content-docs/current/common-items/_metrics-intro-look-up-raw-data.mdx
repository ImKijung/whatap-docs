## メトリクスRaw Dataを表示する

ホーム画面 > プロジェクト選択 > <Cmdname sid="analysis" className="uitext" /> > <Cmdname sid="tag_count" className="uitext" />

メトリクスのRaw Dataを表示するためには、参照基準を選択する必要があります。

<ProdImg img='metrics-chart-example-02.png' desc='' />

- 取得時間 (**必須**)
- 表示するカテゴリ (**必須**)
- 検索するタグ(**選択**、指定されていない場合はすべて)
- 検索するフィールド (**選択**、指定されていない場合はすべて)
