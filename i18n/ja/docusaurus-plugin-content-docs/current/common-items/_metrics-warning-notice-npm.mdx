## テンプレート{#template}

**_アラート通知_** > **_イベント設定_** > **_メトリクス_**セクションで**_+イベント追加_**ボタンを選択してください。 **_テンプレート_**から目的の項目を選択してください。 次のようにアラート通知テンプレートを提供します。

-   **_ネットワークJitterアラート_**

    平均**jitter**が30/msを超えるとアラート通知が送信されます。

-   **_ネットワーク遅延率アラート_**

    平均**latency**が200/msを超えると、アラート通知が送信されます。

-   **_データ転送量アラート_**

    通信**bps**が128Mib/sを超えるとアラート通知が送信されます。
