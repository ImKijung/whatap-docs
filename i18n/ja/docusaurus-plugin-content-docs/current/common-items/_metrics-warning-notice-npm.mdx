## テンプレート{#template}

**_アラート通知_** > **_イベント設定_** > **_メトリクス_**セクションで**_+イベント追加_**ボタンを選択してください。 **_テンプレート_**から希望する項目を選択してください。 次のようにアラート通知テンプレートを提供します。

-   **_ネットワークJitterアラート_**

    平均**jitter**が30/msを超えたらアラート通知を送信します。

-   **_ネットワーク遅延率アラート_**

    平均**latency**が200/msを超えたらアラート通知を送信します。

-   **_データ転送量アラート_**

    通信**bps**が128Mib/sを超えたらアラート通知を送信します。
