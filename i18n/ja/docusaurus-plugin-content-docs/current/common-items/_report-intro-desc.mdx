<InDoc product='browser'>

ホーム画面 > プロジェクト選択 > **_レポート_**

WhaTapモニタリングサービスの初期画面からプロジェクトを選択し、**_プロジェクトメニュー_**の**_レポート_**を選択してください。 

</InDoc>

レポートは、個々のプロジェクトのサービスの使用状況と障害の発生履歴を報告する文書です。サービスモニタリング担当者は、レポートを通じて関連部署の担当者とステータスを共有します。

モニタリングデータ分析は、サービスの改善方向を定める指標となるため、極めて重要です。ただし、複数のダッシュボードからデータを収集して文書化するには煩雑な場合があります。  

WhaTapの**_レポート_**メニューから、**レポート作成業務の自動化**をサポートします。レポートメニューから、週次レポートを作成したり、設定した時間にレポートを共有したり、さまざまな形式で管理することができます。

:::tip

**レポート**をダウンロードする場合、印刷またはメール送信の予約をスケジュールする場合は、そのボタンをクリックしてください。ダウンロードはhtml形式で入手可能です。

:::
