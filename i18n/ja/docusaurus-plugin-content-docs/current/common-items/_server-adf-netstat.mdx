netstatコマンドが入力された場合、1分間隔で収集され、<Cmdname sid="category" className="uitext" />`server_netstat`(netstat connection統計)、`server_netstat_s`(netstat -sコマンドの結果)がアップロードされます。

<Tabs groupId='server-user-script'>
<TabItem value="lxs" label="Linux Shell">

```bash title=SH
echo "netstats.enabled=true" |sudo tee -a /usr/whatap/infra/conf/whatap.conf
```

</TabItem>
<TabItem value="wps" label="Windows Powershell">

```bash title=SH
Add-Content -Path "C:\Program Files\WhatapInfra\whatap.conf" -Value "netstats.enabled=true"
```

</TabItem>
</Tabs>
