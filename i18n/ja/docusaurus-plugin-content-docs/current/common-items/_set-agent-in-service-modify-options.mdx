### オプションを修正・削除する

<ProdImg img='modify-agent.png' desc='オプションを修正' />

画面を上下にスクロールするか、左側のオプションカテゴリを選択して修正または削除するオプションを選択してください。画面の上部に移動するには、**_オプション追加へ移動_**ボタンを選択してください。

変更するオプションで目的の値を選択または修正してください。オプションを削除するには![削除アイコン](/img/ico-trash.svg)ボタンを選択してください。変更を適用するには、**_適用_**ボタンを選択してください。

:::note

オプション値に何も入力しない状態で**_適用_**ボタンを選択した場合は、オプションを削除します。

:::
