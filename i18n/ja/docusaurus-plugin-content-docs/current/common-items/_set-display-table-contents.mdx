1. テーブル右上の ![設定アイコン](/img/ico-setting.svg)ボタンを選択してください。

2. ドロップダウンメニューから<Cmdname sid="table_content_setting" className="uitext" />を選択してください。

3. 次のオプションを設定してください。

   - <Cmdname sid="whether_display_content" className="uitext" />：テーブル一覧に表示するコンテンツの表示有無を選択してください。

   - <Cmdname sid="tag_management" className="uitext" />：テーブル一覧に表示したいタグを選択できます。 見えないように設定したり、手順を変更したりできます。

4. すべての設定を完了して ![閉じるアイコン](/img/ico-close.svg) ボタンを選択してください。

設定した内容がテーブルに適用されます。
