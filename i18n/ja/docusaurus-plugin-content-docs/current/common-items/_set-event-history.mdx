:::note

選択したプロジェクトによって、画面イメージが異なる場合があります。

:::

-   <Cmdname sid="time_selector" className="uitext" />

    -   右側の緑色のボタンを選択して表示時間を選択することができます。

    -   ![左アイコン](/img/left-arrow.svg)または ![右アイコン](/img/right-arrow.svg)ボタンを選択すると、選択した表示時間だけ間隔を移動します。

    -   詳細な時間を選択するには、日付または時間の領域を選択してください。詳細な時間を設定し、<Cmdname sid="apply" className="uitext" />ボタンを選択します。

-   <Cmdname sid="filter" className="uitext" />: <Cmdname sid="title" className="uitext" />または<Cmdname sid="message" className="uitext" />内容に基づいてイベント履歴をフィルタリングできます。

-   <Cmdname sid="all_selected" className="uitext" />：プロジェクトに含まれるエージェントを選択できます。

-   **_CSV_**：参照したイベント履歴結果をcsvファイルとして保存することができます。<Cmdname sid="maximum_csv_lines" className="uitext" />を入力し <Cmdname sid="download" className="uitext" /> ボタンを選択してください。

-   <Cmdname sid="column_select" className="uitext" />：取得されたイベント履歴の結果の列項目を追加することができます。

-   <Cmdname sid="event_setting" className="uitext" />：<Cmdname sid="event_statistic" className="uitext" /> > <Cmdname sid="event_setting" className="uitext" />メニューに移動します。
