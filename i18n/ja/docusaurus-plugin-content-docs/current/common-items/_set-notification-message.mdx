ホーム画面 > プロジェクト選択 > **_アラート通知_** > **_イベント設定_** >**_メトリクス_**タブ選択

プロジェクトメンバーにメールまたはサードパーティープラグインを通じて伝える、アラート通知メッセージを修正できるユーザー定義機能を提供します。 画面の右上の**_通知メッセージ設定_**ボタンを選択してください。 受信するメッセージで必要でないと思われる項目を隠すことも、ユーザーが希望する特別な内容を追加することもできます。

<ImgLang img='notification-set-message.png' desc='알림 메시지 설정' />

:::note

- この機能は**メトリクス**、**複合メトリクス**アラート通知のみに適応できます。

- この機能はプロジェクト選択>**_サイトマップ_**>**_ラボラトリー_**>**_通知メッセージ設定_**メニュー経路でも進入できます。

- この機能は**アラート設定**権限を持つメンバーが設定できます。 詳しい内容は[次の文書](../project/project-structure#project-auth)を参照してください。

:::

## 基本画面ガイド

<ImgLang img='notification-set-message-main.png' desc='알림 메시지 설정' />

- アラートメッセージの種類は<span class='occur set-noti'>発生</span>と<span class='resolve set-noti'>解決</span>で分類できます。 <span class='occur set-noti'>発生</span>はイベントが発生した時に届く通知であり、<span class='resolve set-noti'>解決</span>はイベントが解決できた時に届く通知です。

- ![数字 1](/img/number-01.png)領域は受信するメッセージの詳細項目です。 ![数字 2](/img/number-02.png)領域は受信するメッセージの詳細項目に対する内容です。 ![数字 3](/img/number-03.png) ![プレビューアイコン](/img/ico-noti.svg)**_プレビュー_**ボタンを選択すると、受信するメッセージの内容を予め確認できます。

- 受信するメッセージの内容を修正するには![数字 4](/img/number-04.png) ![編集アイコン](/img/ico-edit.svg)**_修正_**ボタンを選択してください。

- 受信したメッセージの詳細項目を隠すには![数字 5](/img/number-05.png)トグルボタンを選択して非アクティブにしてください。

## 受信メッセージ詳細項目を隠す

受信メッセージの内容で、必要でないと思われる項目を隠すことができます。 各詳細項目の一番右のトグルボタンを選択して、該当項目を受信メッセージで隠すことも、表示することも設定できます。 ![プレビューアイコン](/img/ico-noti.svg)**_プレビュー_**ボタンを選択すると、受信するメッセージの内容を予め確認できます。

- 詳細項目を活性化する例

  <ImgLang img='notification-set-message-preview.png' desc='활성화 미리보기' />

- 詳細項目を非活性化する例

  <ImgLang img='notification-set-message-preview-disabled.png' desc='비활성화 미리보기' />

変更事項を反映するには![保存アイコン](/img/ic-s-check.svg)**_保存_**ボタンを選択してください。 他のカテゴリーの変更事項を一挙に反映するには、画面右上の**_すべて保存_**ボタンを選択してください。

:::note

- 隠した詳細項目を受信メッセージにまた追加するには、該当項目のトグルボタンを選択して活性化してください。

- すべての詳細項目を受信メッセージに追加するには![活性化アイコン](/img/ic-power.svg)**_すべて活性化_**ボタンを選択してください。

:::

## 詳細項目内容を修正する

詳細項目の内容でユーザーが希望する特別な内容を追加できます。 

1. 各詳細項目の一番右の![編集アイコン](/img/ico-edit.svg)**_修正_**ボタンを選択してください。

2. **_メッセージ編集_**ウィンドウが表示されたら、追加内容を入力してください。

   <ImgLang img='notification-set-message-edit.png' desc='메시지 편집' />

3. **_確認_**ボタンを選択してください。

4. ![プレビューアイコン](/img/ico-noti.svg)**_プレビュー_**ボタンを選択すると、受信するメッセージの内容を予め確認してください。

   <ImgLang img='notification-set-message-edit-preview.png' desc='메시지 편집' />

変更事項を反映するには![保存アイコン](/img/ic-s-check.svg)**_保存_**ボタンを選択してください。 他のカテゴリーの変更事項を一挙に反映するには、画面右上の**_すべて保存_**ボタンを選択してください。

:::note

**_メッセージ編集_**ウィンドウでテキストブロックはアラート発生状況によって、変更され出力される変数です。 テキストブロックはドラッグして位置を変更できます。 テキストブロックを削除した場合、ウィンドウ下方のテキストブロックボタンを選択して、再度追加できます。

<ImgLang img='notification-set-message-edit-block.png' desc='메시지 편집' />

:::

:::caution

ユーザーによって内容を修正する前のメッセージは、ユーザーの言語状態によって翻訳され転送されますが、内容の修正後は言語の状態に関わらず、ユーザー定義されたフォーマットのみで通知を転送します。

:::

## 他のプロジェクトに設定をコピーする

ユーザーが設定した内容をほかのプロジェクトにコピーできます。 

1. 画面右上で**_他のプロジェクトにコピー_**ボタンを選択してください。

2. **_プロジェクト適用_**ウィンドウが表示されたら、設定内容をコピーするプロジェクトを選択してください。 すべてのプロジェクトを選択するには**_すべて選択_**を選択してください。

3. **_適用_**ボタンを選択します。

:::info

現在のプロジェクトと同一の製品タイプのみ選択できます。

:::

## 初期化

ユーザーが設定したすべての変更事項を初期化するには![初期化アイコン](/img/ico-reset.svg) **_初期化_**ボタンを選択してください。 すべてのカテゴリーの変更事項を初期化するには、画面右上の**_すべて初期化_**ボタンを選択してください。 確認メッセージのウィンドウが表示されたら**_確認_**ボタンを選択してください。

:::caution

初期化された設定は復旧できません。

:::
