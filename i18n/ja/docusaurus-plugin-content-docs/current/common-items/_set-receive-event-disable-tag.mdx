### イベント受信タグを解除する

1. <Cmdname sid="alert_receive_settings_by_member" className="uitext" />セクションのユーザー一覧の一番左の<LinkImage img="ico-add.svg" desc="詳細アイコン" />ボタンを選択してください。

2. <Cmdname sid="event_tag" className="uitext" />ポップアップウィンドウが表示されたら、適用されたタグの横の<LinkImage img="ico-close.svg" desc="닫기 아이콘" />ボタンを選択してください。

3. <Cmdname sid="event_tag" className="uitext" />ポップアップウィンドウを閉じてください。

イベント受信タグを解除します。
