## 受信レベルを選択する

アラート通知レベルに応じて、通知の受信有無を選択できます。<Cmdname sid="alert_receive_settings_by_member" className="uitext" />セクションのユーザー一覧から<Cmdname sid="receive_level" className="uitext" />に<Cmdname sid="all" className="uitext" />または<Cmdname sid="fatal" className="uitext" />を選択してください。

- <Cmdname sid="all" className="uitext" />：すべてのアラート通知を受信できます。

- <Cmdname sid="fatal" className="uitext" />：重大レベルのアラート通知のみ受信できます。
