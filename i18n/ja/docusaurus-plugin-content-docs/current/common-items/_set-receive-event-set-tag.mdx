## チーム別・ユーザー別の通知を設定する

プロジェクトのメンバーのうち、特定メンバーまたはチームに対する通知の受信有無を設定します。**_メトリクス_**、**_異常値検知_**、**_ログ_**イベント設定の**_イベント受信タグ_**項目でタグを追加してください。 イベントごとに警告通知を受信するメンバーまたはチームを選択できます。

逆に、イベント受信タグを設定しないと、全体メンバーに警告通知を送信できます。

<details>
<summary>イベント受信タグの使用例</summary>

イベント受信タグの使用例を使用して、チーム別またはユーザー別によるアラート通知を送信する設定方法について説明します。次のように、チームごとに異なるアラート通知を設定します。

| 通知の種類             |           サーバーチームの受信有無          |          DevOpsチームの受信有無         |
| ----------------- | :-----------------------------: | :-----------------------------: |
| メトリックアラート通知       | <span class='emoji-ok'>✅</span> | <span class='emoji-no'>❌</span> |
| 異常値検知の警告通知        | <span class='emoji-ok'>✅</span> | <span class='emoji-ok'>✅</span> |
| ログのリアルタイム警告通知     | <span class='emoji-no'>❌</span> | <span class='emoji-ok'>✅</span> |

1.  **イベント受信の設定** (**_警告通知_** > **_イベント受信設定_**)

    <ImgLang img='set-event-tag-01.png' desc='ユーザー別イベント受信設定' />

    プロジェクトに属するメンバーのうち、サーバーチーム所属は`サーバーチーム`、DevOps所属は`DevOpsチーム`とイベント受信タグを設定してください。

2.  **メトリクス警告通知の設定** (**_警告通知_** > **_イベント設定_** > **_メトリクス_** > **_+ イベント追加_** > **_イベント受信設定_**)

    <ImgLang img='set-event-tag-02.png' desc='ユーザー別イベント受信設定' />

    **_+ タグ追加_**または**_+_**ボタンをクリックしてください。タグ一覧でご希望のタグを選択するか、新しいタグを作成してください。メトリクスイベントに関する通知を`サーバーチーム`に設定した場合です。

3.  **_ログのリアルタイム警告通知設定_** (**_警告通知_** > **_イベント設定_** > **_ログ_** > **_+ イベント追加_** > **_イベント受信タグ_**)

    <ImgLang img='set-event-tag-03.png' desc='ユーザー別イベント受信設定' />

    **_+ タグ追加_**または**_+_**ボタンをクリックしてください。タグ一覧でご希望のタグを選択するか、新しいタグを作成してください。ログのリアルタイムイベントに関する通知を`DevOpsチーム`に設定した場合です。

4.  異常値検知イベントは、全体メンバーに警告通知を送信するため、イベント受信タグを設定しません。

</details>
