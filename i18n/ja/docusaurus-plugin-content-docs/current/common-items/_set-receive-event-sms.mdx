### SMS通知受信を追加する

SMS通知受信が必要な場合、**_アカウント情報管理_**でユーザーの電話番号を設定してください。 

<ImgLang img='set-event-sms-notification.png' desc='ユーザー電話番号' />

1.  画面の右上のプロファイルアイコンを選択してください。 

2.  ポップアップメニューが表示されたら、**_アカウント情報管理_**ボタンをクリックしてください。 

3.  **_ユーザー電話番号_**セクションで**_通常の携帯電話_**ボタンを選択してください。 

4.  **_電話番号_**にSMSを受信する電話番号を入力してください。

5.  **_SMS認証_**ボタンを選択してください。

6.  **_SMS認証_**ウィンドウが表示されると、ユーザーの携帯電話に送信された認証コードを入力してください。

7.  **_電話番号変更_**ボタンを選択してください。

:::note

-   登録した電話番号を変更するには、**_番号を変更する_**ボタンを選択し、上記の手順を繰り返してください。

-   SMSを通知で受信できる電話番号は、**韓国の携帯番号**のみ登録できます。

:::
