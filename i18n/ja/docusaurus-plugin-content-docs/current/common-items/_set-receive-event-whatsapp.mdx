### WhatsApp通知受信を追加する

WhatsApp経由で通知を受信できます。

<ImgLang img='set-event-whatsapp-notification.png' desc='WhatsApp'/>

1. 画面の右上のプロファイルアイコンを選択してください。 

2. ポップアップメニューが表示されたら、**_アカウント情報管理_**ボタンをクリックしてください。 

3. **_ユーザー電話番号_**セクションで**_WhatsApp_**ボタンを選択してください。 

4. **_電話番号_**にSMSを受信する電話番号を入力してください。

5. **_認証番号送信_**ボタンを選択してください。

6. WhatsAppアプリケーションに送信された認証番号6桁を入力してください。

7. **_認証する_**ボタンを選択してください。

:::note

登録した電話番号を変更するには、**_番号を変更する_**ボタンを選択し、上記の手順を繰り返してください。

:::
