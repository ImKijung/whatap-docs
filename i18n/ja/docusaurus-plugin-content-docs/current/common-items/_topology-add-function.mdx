トポロジーメニューから使用できるアドオンについて説明します。

## ハイライトフィルター

特定ノードを選択し、関連づけられているノードを強調して表示するための機能を提供します。大規模なノードとトポロジー間の複雑な関連付けを簡単に確認できます。

<ImgLang img='topology-highlight.png' desc='ハイライトフィルター' />

-   特定のノードにカーソルを合わせると、そのノードや関連づけられたノードおよびリンク情報がハイライトされ表示します。
-   キーワード入力により、ノードの選択を保持できます。トポロジー更新時もハイライト状態が維持されます。

## ノード選択機能の利用

**_ノード選択_**ボタンを選択してください。画面右側のプロジェクト、DB呼び出し、外部呼び出しノードをグルーピングしてダイアグラムで提供します。 

<ImgLang img='topology-select-node.png' desc='ノード選択' />

右側のダイアグラムで下位ノードを選択した場合は、左側のトポロジー画面でハイライトフィルターを適用します。

<ImgLang img='topology-select-node-agent.png' desc='ノード選択' />

右側の画面では、次の機能を使用できます。

-   ズームイン/ズームアウト
-   ドラッグ
-   下位ノードのクリック時のハイライトフィルターの適用
-   上位ノードのクリック時の下位ノードの表示または非表示

ハイライトフィルター機能をキャンセルするには、キーワード入力欄のエージェント名を削除してください。

:::note

**_グループ統計トポロジー_**メニューでは、画面右下の**_ノード選択(ダブルクリック)_**ボタンによって機能が提供されます。

:::

## コンテキストメニュー

トポロジーのノードを右クリックすると、コンテキストメニューが表示されます。コンテキストメニューには、アプリケーションモニタリングダッシュボードに移動するためのリンクなどのメニューを提供します。

<ImgLang img='context_menu.png' desc='コンテキストメニュー' />

:::note

**_コンテナマップ_**、**_ノードリソース_**、**_コンテナリソース_**メニューはコンテナモニタリングに限って提供します。

:::

## ノード位置の保存と復元

トポロジー画面では、チャートのノード位置をマウスで左クリックした状態でドラッグすると変更できます。変更したノードの位置は、別のメニューに入ってからトポロジ画面に戻った場合に維持されます。

-   **_保存位置_**：ユーザーが変更したノードの位置情報を保存できます。
-   **_復元位置_**：ユーザーが保存位置を実行した時点でノードの位置を復元します。
-   **_初期化の位置_**：ノードの位置を初期化します。

:::note

-   ノードロケーションメニューは、ユーザーが入力するトポロジーメニューによっては対応されない場合があります。
-   変更したノードの位置情報は、ウェブブラウザーのローカルストレージに保存されます。他のユーザーの画面では、変更されたノードの位置は反映されません。

:::

## 別名の付与と除去

1.  **_ノード選択_**ボタンを選択してください。

2.  画面の右側で、別名を設定するノードに移動し、マウス右クリックをしてください。

    ![別名入力](/img/node-custom-name.png)

3.  別名入力ウィンドウが表示されたら、別名を入力してEnterキーを入力してください。

別名情報を保存します。チャートのデータを新たに参照する時点で、保存した別名をトポロジー画面のノードに適用します。

:::note

ノードの別名情報は、ウェブブラウザのローカルストレージに保存されます。

:::

## しきい値超過情報の強調

ノード間の呼び出しレスポンス時間およびエラー件数のしきい値を設定する場合は、しきい値を超えたノードおよび呼び出し関係を強調して表示します。トポロジー画面のメニューで**_アラート表示_**ボタンを選択してください。**_レスポンス時間(ms)_**と**_エラー件数_**テキスト入力ボックスに必要な設定値を入力してください。

<ImgLang img='threshold_emphasized.png' desc='しきい値を超過する情報を強調' />

:::note

**_アプリケーショントポロジー_**メニューから**_アラート表示_**を選択した後、設定値を入力できます。

:::
