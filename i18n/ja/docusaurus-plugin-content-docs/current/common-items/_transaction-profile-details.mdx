## 詳細分析{#details}

ホーム画面 > プロジェクト選択 > **_分析_** > **_ヒートマップ_**

<ImgLang img='tx-hitmap.png' desc='히트맵' />

1.  ![数字1](/img/number-01.png)**_時間選択_**で、ヒートマップを照会する期間を選択してください。

2.  ![数字1](/img/number-01.png)**_区分_**、**_アプリケーション_**から照会対象を選択してください。

3.  ![検索アイコン](/img/ico-btn-search.png)ボタンを選択してください。

4.  ![数字2](/img/number-02.png)**_ヒートマップトランザクション_**から目的の領域を![数字3](/img/number-03.png)のようにドラッグしてください。

5.  ![数字5](/img/number-05.png)**_TXトレース_**に一覧が表示されたら、分析したいトランザクション項目を選択してください。

トランザクションのパフォーマンス分析のためのクライアント情報などの属性、トランザクションの処理パフォーマンス、各区間別詳細遂行履歴などを確認できる**_トランザクション情報_**ウィンドウが表示されます。

:::note

-   **_ヒートマップ_**メニューの機能の詳細については、[次の文書]](trs-view#details)を参照してください。

-   アプリケーションダッシュボードのヒートマップウィジェットの詳細については、[次の文書]](dashboard-hitmap-trace)を参照してください。

:::
