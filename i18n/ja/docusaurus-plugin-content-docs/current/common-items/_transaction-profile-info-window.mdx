### トランザクション情報ウィンドウのご案内{#transaction-info}

<ImgLang img='transaction-info-window.png' desc='トランザクション情報' />

-   ![数字1](/img/number-01.png)領域では、ステップ情報をさまざまな形式で確認できます。 希望する形式のタブを選択してください。

-   ![数字2](/img/number-02.png)領域では、トランザクションに関する基本情報を確認できます。

-   ![数字3](/img/number-03.png)![統計アイコン](/img/ic-static.svg):該当URLまたは各実行区間の統計データウィンドウが表示されます。 統計データウィンドウで**_詳細>_**ボタンを選択すると**_統計/レポート_**>**_統計_**メニューに移動します。

    :::note

    **_統計_**メニューの詳細については、[次の文書](analysis-apm-trs)を参照してください。

    :::

-   ![数字4](/img/number-04.png)SQLステップを選択すると、SQL変数とHTTPクエリを照会することができます。 詳細については、[次の文書](#transaction-parameter)を参照してください。
