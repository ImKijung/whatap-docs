.NETアプリケーションサーバーのWeb Serviceを追跡できます。

- **webservice_method_enabled** <span class='type'>Boolean</span>

  既定値`false`

  値を`true`に設定するとWeb Serviceを追跡できます。 

  :::note

  設定した値を有効にするには、IISを再起動してください。

  :::

- **webservice_method_prefix** <span class='type'>String</span>

  追跡するメソッド名をネームスペースを入力してください。 Web Serviceは直接追跡できないため、Web Serviceを使用したメソッドを直接追跡対象として入力する必要があります。

  次の例を参照にしてください。

  ```ini title='whatap.conf'
  webservice_method_prefix=Test.Demo, WebServiceDemo.WebService.
  ```

  :::caution

  - 条件を満たすメソッドは、Web Serviceを使用していない場合でも追跡されます。 ただし、Web Serviceを使用するメソッドでなければ、エラーが発生してもエラーは検出されず、トランザクションが終わっていない状態に維持される可能性があります。

  - メソッドが所属するクラスまで入力すると、そのクラス内のすべてのメソッドが追跡対象になります。

  :::

  :::note

  設定した値を有効にするには、IISを再起動してください。

  :::
