Focusは、WhaTapモニタリングで提供されていない任意のデータを時系列でアップロードするプログラムです。 

運用チームが自社開発したチェックスクリプトやデータベースクエリを定期的または一回のみで動作して収集された値をアップロードできます。アップロードされた値は、そのプロジェクトの分析・メトリクスチャートメニューで確認できます。また、統合メトリクスダッシュボードに追加できます。

Focusを使用するには、WhaTapプロジェクトの中でServer、Applicatoon(Java、PHP、NodeJS、Python)、Database、Kubernetesプロジェクトのいずれかを選択した後に、プロジェクトアクセスキー文字列と収集サーバーIPを実行オプションとして指定して使用できます。
