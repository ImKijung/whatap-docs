Web Application Server(WAS)、バッチアプリケーションなど、JVM環境で動作するすべてのアプリケーションに適用できます。アプリケーションの障害分析のためにThread Dump(スレッドダンプ) を保護する代わりに、既に確保されたスタックを使用して障害状況を再現することなく、問題の原因を正確に特定できます。
