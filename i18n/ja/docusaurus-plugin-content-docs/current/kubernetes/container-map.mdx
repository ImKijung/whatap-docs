WhaTap Kubernetesの**_コンテナマップ_**は、Kubernetesモニタリングの開始点です。ユーザー設定の表示によって、各オブジェクトは1つのコンテナ、または1つのPodを表示します。

次の動画は、**_コンテナマップ_**メニューに関する基本ガイドです。

{@youtube: GHOJW3Rk3jc}

## 基本画面ガイド

![コンテナマップ](/img/kube-container-map-01.png)

:::info

**ブロック(Block)とマイクロブロック(Micro block)**

コンテナマップでは、1つの多角形をブロック(Block)と言います。このとき、エージェントを含むブロック(ブロック中のブロック)をマイクロブロック(Micro Block)と言います。

:::

### ![](/img/number-01.png)コンテナまたはPod

各オブジェクトの外観とデザインは、コンテナーまたはポッドの特性と状態を表します。

-   色：コンテナまたはPodの状態を意味します。状態の詳細については、[次の文書](container-status)を参照ください。

-   枠線

    -   オブジェクトの中には、境界線が表示されます。これはコンテナに、Java、Node.js、Pythonなどのマイクロエージェントをインストールする場所です。 

    -   1つのPod内に2つのコンテナがあり、2つのコンテナのうち１つだけにマイクロエージェントをインストールした場合、境界線が表示されます。つまり、Pod内のコンテナのいずれか一つでもマイクロエージェントがインストールされている場合は、境界線が表示されます。

### ![](/img/number-02.png)クラスタープロジェクト - ネームスペースプロジェクトの移動

![コンテナマップのプロジェクト一覧](https://img.whatap.io/media/images/whatap_kubernetes_container_map_namespace_project_list.png)

クラスタープロジェクトの下にネームスペースプロジェクトを作成した場合、コンテナマップの画面上にあるプロジェクト一覧ボックスを使用してクラスタープロジェクトを含むプロジェクト一覧の確認およびプロジェクト間を移動できます。

:::note

-   WhaTap Kubernetesサービスの**クラスタープロジェクト**は、**マスタープロジェクト**の新しい名前です。

-   ネームスペースプロジェクトの作成に関する詳細は、[次の文書](namespace)を参照してください。

:::

### ![](/img/number-03.png)ユーザー設定

![ユーザー設定](/img/container-map-custom-view.png)

コンテナマップには、複数のユーザー設定が用意されており、目的の画面表示を作成する機能が画面順に用意されています。

-   カテゴリ：**_Container_**または**_Pod_**を選択することができます。

-   **_グループ化_**(クラスタリング)：グループ化できる条件を選択できます。コンテナまたはPodを選択した条件に応じてカードで区切って表示します。詳細は[次の文書](container-group)を参照してください。

-   **_ConState(state)_**：チャート指標を選択できます。

-   ![設定アイコン](/img/ico-setting.svg)ボタンをクリックすると、各オブジェクトにコンテナまたはPodの情報を表示できるオプションを選択できます。

    ![オプション](/img/k8s-container-map-setting.png)

    -   **_WhaTapエージェントの表示_**：コンテナまたはPodの情報を表示できます。必要な**_ラベル_**オプションを選択します。ラベルオプションの詳細については、[次の文書](#label)を参照してください。

    -   **_カードの下部にメトリクスを表示する_**：画面下部にメトリクス情報を表示するかどうかを選択できます。

    -   **_1分以内に作成されたコンテナを表示する_**：作成されてから1分以内のコンテナの場合、明るい色で表示するオプションです。

-   ![フィルターアイコン](/img/ico-filter.svg) **_フィルター_**：ユーザーが選択した項目に基づいてフィルタリングし、コンテナまたはPodの状態を確認できます。

-   ![リセットアイコン](/img/ico-reset.svg) **_リセット_**：ユーザーが設定した画面表示を初期化します。

#### ラベルオプション{#label}

-   **_なし_**：ブロックにラベルを表示しません。

-   **_名前_**：コンテナ/PODの名前を表示します。

-   **_アプリケーション_**：コンテナ/POD内のアプリケーションを表示します。

-   **_選択された指標_**：**_ConState(state)_**項目で選択されたチャート指標を表示します。

### 概要ビューと詳細ビュー

ユーザーは、関心のあるコンテナまたはPodを選択して概要および詳細情報を確認できます。この情報には、次のような主要なモニタリングデータが含まれます。

-   **트레이스** (Trace)

    컨테이너 상의 애플리케이션 추적을 위한 APM 도구를 의미합니다. 

-   **메트릭스** (Metrics)

    노드, 쿠버네티스의 다양한 지표 데이터를 의미합니다. 

-   **로그** (Log)

    컨테이너, 애플리케이션의 각종 로그를 의미합니다.

-   **이벤트** (Event)

    쿠버네티스 이벤트, 사용자 정의 이벤트를 의미합니다. 

これらのデータに基づいて、ユーザーはKubernetes環境のObservabilityを得ることができます。以下の**概要を見る**、**詳細を見る**画面は、いずれも上記の4項目を中心とした情報を提供します。

#### 概要を表示

特定のオブジェクトまたはオブジェクトのセットをクリックします。右側に表示される**概要を見る**画面を通じて該当オブジェクトの概要を確認することができます。

![概要を見る](https://img.whatap.io/media/images/whatap_kubernetes_container_map_right_summary.png)

-   **_트레이스_**

    컨테이너 맵의 **_트레이스_** 탭을 통해 컨테이너 상에서 실행되는 애플리케이션의 개별 트랜잭션을 추적할 수 있습니다.

-   **_メトリクス_**

    컨테이너 맵의 **_메트릭스_** 탭을 통해 각 컨테이너의 자원 할당량, 사용량 등의 각종 지표 데이터를 실시간으로 확인할 수 있습니다.

-   **_ログ_** 

    컨테이너 맵의 **_로그_** 탭을 통해 각 컨테이너의 표준 출력 로그 및 컨테이너 상의 애플리케이션이 생성하는 로그를 통합 관리, 조회할 수 있습니다.

-   **_イベント_** 

    컨테이너 맵의 **_이벤트_** 탭을 통해 쿠버네티스 클러스터에서 발생한 이벤트 및 와탭의 경고 기능을 이용한 사용자 임계치 기반 이벤트의 통합 조회가 가능합니다. 

#### 詳細を見る

**요약 보기** 화면의 오른쪽 위에 ![상세 보기 아이콘](/img/ico-detail-mode.svg) 버튼을 클릭하세요. **상세 보기** 화면으로 전환합니다. **상세 보기** 화면 구성에 대한 자세한 안내는 [다음 문서](container-map-detail-mode)를 참조하세요.

![詳細を見る](/img/container-map-detail-mode.png)

### リアルタイム及び履歴照会

コンテナマップは、基本的にリアルタイム参照ベースですが、複数の時点に基づいて参照機能を提供します。リアルタイムの変更は、画面右上の時間セレクタを使用します。

![時間セレクタ](/img/kubernetes-time-seletor.png)

| 追跡ステータス             | 説明                                                                 |
| ------------------- | ------------------------------------------------------------------ |
| リアルタイム参照（LIVE）      | 基本モード<br/>10秒間隔でリアルタイムステータス変更                                      |
| 一時停止の照会 (PAUSE)     | リアルタイムステータス変更OFF<br/>現在のステータスを詳しく確認するために使用                         |
| 過去の時点参照             | ユーザーは、過去の特定の時点を指定して確認<br/>年/月/日/時/分を指定できます                         |
