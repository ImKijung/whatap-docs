ホーム画面 > プロジェクト選択 > **_役人_** > **_ネームスペース管理_**

クラスタープロジェクトでは、**_ネームスペース管理_**メニューを通じてネームスペースプロジェクトを管理することができます。ネームスペースプロジェクトでは、そのメニューに進入することはできません。

:::note

クラスタープロジェクトの**_ダッシュボード_** > **_コンテナマップ_**メニュー画面の上には、下位ネームスペースまたは**_ネームスペース現況_**および**_ネームスペース管理_**メニューに移動できるリストボックスを提供します。 

![ネームスペース移動](/img/k8s-namespace-project-list-check.png)

:::

## ネームスペースプロジェクト管理

![ネームスペース管理](/img/k8s-namespace-mng.png)

上段の![number-1](/img/number-01.png)領域で、全体対象数および主要なリソースの数を確認できます。![number-2](/img/number-02.png) **_ネームスペース一覧_**領域でネームスペースプロジェクトを照会し、追加および削除できます。

### ネームスペースプロジェクトの作成

**_ネームスペース一覧_**でプロジェクトを作成する項目の**_+ プロジェクト追加_**ボタンを選択してください。**_プロジェクト追加_**ウィンドウが表示されたら**_確認_**ボタンを選択してください。

### ネームスペースプロジェクトの削除

1.  **_ネームスペース一覧_**で削除しようとするプロジェクト項目の一番右側の![設定アイコン](/img/ico-setting.svg) **_設定_**アイコンを選択してください。

2.  **_プロジェクト管理_**メニューに移動してください。

3.  画面右上の**_プロジェクト削除_**ボタンを選択してください。

:::caution

ネームスペースプロジェクトを削除する場合、ユーザー一覧情報とこれまで収集したデータはすべて削除されます。削除したデータは再復旧できませんので、注意してください。

:::

:::note

プロジェクト管理の詳細については、[次の文書](../project/project-manage)を参考にしてください。

:::
