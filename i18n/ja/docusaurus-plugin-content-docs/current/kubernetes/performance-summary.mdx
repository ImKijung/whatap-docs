ホーム画面 > プロジェクトを選択 > **_リソース_** > **_パフォーマンスの概要_**

最近の時間別チャートによって、コンテナ/Pod、Node数などのKubernetes主要な指標がどのように変更されたかを確認できます。

![パフォーマンス概要](/img/performance-summary.png)

ノード数量ウィジェット右上に![その他のアイコン](/img/ico-more.svg)**_その他の_**アイコン>![ショートカットアイコン](/img/ico-shortcut.svg)**_ショートカット_**アイコンを順番に選択すると**_リソース_**>**_ノード一覧_**メニューに移動します。 

ウィジェットを提供するNode情報は次のとおりです。

-   直近30分チャート
-   **_Node CPU (%)_**、**_Node Memory (%)_**など

ノードCPUウィジェットとメモリウィジェットの右上に![詳細を見るアイコン](/img/ico-more.svg)**_もっと見る_**アイコン>![ショートカットアイコン](/img/ico-shortcut.svg)**_ショートカット_**アイコンを順番に選択すると**_分析_**>**_メトリクスチャート_**メニューに移動します。このメニューを使用して、より詳細な条件でチャートを確認できます。

ウィジェットによって提供するコンテナ情報は次のとおりです。

-   直近10分チャート
-   **_Container Status_**、**_Container CPU (%)_**、**_Container Memory (%)_**など

ウィジェットによって提供するアプリケーション情報は次のとおりです。

-   直近10分チャート
-   **_Active TX_**、**_TPS_**、**_Response Time_**など

:::note

アプリケーション情報は、アプリケーションエージェントのインストール対象が存在する場合に表示します。

:::
