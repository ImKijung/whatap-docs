ホーム画面 > プロジェクト選択 > **_リソース_** > **_パフォーマンス要約_**

最近の時間別チャートを通じてコンテナ/Pod数量、Node数量などクKubernetes主要指標にどのような変化があったかを確認することができます。

![パフォーマンス要約](/img/performance-summary.png)

ノード数量ウィジェット右上に![もっと見るアイコン](/img/ico-more.svg)**_もっと見る_**アイコン>![ショートカットアイコン](/img/ico-shortcut.svg)**_ショートカット_**アイコンを順番に選択すると**_リソース_**>**_ノード一覧_**メニューに移動します。 

ウィジェットとして提供するNode情報は次のとおりです。

-   直近30分チャート
-   **_Node CPU (%)_**、**_Node Memory (%)_**など

ノードCPUウィジェットとメモリウィジェットの右上に![詳細を見るアイコン](/img/ico-more.svg)**_もっと見る_**アイコン>![ショートカットアイコン](/img/ico-shortcut.svg)**_ショートカット_**アイコンを順番に選択すると**_分析_**>**_メトリクスチャート_**メニューに移動します。このメニューを使用すると、より詳細な条件でチャートを確認できます。

ウィジェットとして提供するコンテナ情報は次のとおりです。

-   直近10分チャート
-   **_Container Status_**、**_Container CPU (%)_**、**_Container Memory (%)_**など

ウィジェットとして提供するアプリケーション情報は次のとおりです。

-   直近10分チャート
-   **_Active TX_**、**_TPS_**、**_Response Time_**など

:::note

アプリケーション情報は、アプリケーションエージェントのインストール対象が存在するときに表示します。

:::
