ホーム画面 > プロジェクト選択 > **_リソース_** > **_マスターメタ情報_**

Kubernetesマスターのメタ情報が一目で確認できます。

<ImgLang img='whatap_kubernetes_resource_master.png' desc='マスターメタ情報' />

以下は、提供する主要なメタ情報です。

| 分類             | 対象                                                                                            |
| -------------- | --------------------------------------------------------------------------------------------- |
| Workload       | Event<br/>CronJob<br/>DaemonSet<br/>Deployment<br/>Job<br/>Pod<br/>ReplicaSet<br/>StatefulSet |
| Service        | Ingresss<br/>Service                                                                          |
| Config/Storage | ConfigMap<br/>Persistent Volume<br/>PersistentVolumeClaim<br/>StorageClass                    |
| Cluster        | ClusterRoleBinding<br/>ClusterRole<br/>Node<br/>Role<br/>ServiceAccount                       |

:::note

マスターメタ情報が参照できない場合は、Kubernetes環境のWhaTapマスターエージェントが正常に動作しているか確認してください。

:::
