WhaTap監視サービスの無料試用期間を過ぎた場合、またはサービスの無料利用基準を超えた場合は、決済情報を登録し、有料サービスに切り替える必要があります。

## 決済情報の登録{#register-pay}

お支払い方法は、カードの定期購入またはインボイス対応請求書の2種類からお選びいただけます。 決済を進める前に、次の事項を事前に確認してください。

-   **決済**権限アカウント

-   カード決済の場合、決済カード番号及び有効期限

-   タックスインボイス、事業者登録証、請求書受領メールの発行の場合

:::note

**決済**権限の詳細については、[次の文書](../project/project-structure#project-auth)を参照してください。

:::

### クレジットカード

1.  [WhaTapモニタリングサービス](https://service.whatap.io)で**決済**権限を持つアカウントでログインしてください。

2.  ホーム画面の左下にある**_利用履歴_**を選択してください。

    <ImgLang img='tax_invoice_1.png' desc='利用履歴' />

3.  ***決済情報***ページへ移動すると、**_決済情報登録_**ボタンをクリックしてください。

4.  **_決済情報の登録_**ウィンドウが表示されたら、まず国を選択してください。

5.  会社名または法人名を入力してください。

6.  **_電話番号_**と**_請求書の言語_**項目を確認し、必要に応じて変更してください。

7.  所属部署名を入力してください。 これは必須ではありません。

8.  **_次へ_**ボタンを選択してください。

#### 国内ユーザー

1.  決済方法を<Cmdname sid="credit_card_iamport" className="uitext" />で選択し、<Cmdname sid="next" className="uitext" />ボタンを選択してください。

2.  クレジットカードのカード番号、有効期限、パスワードの最初の2桁の数字を順番に入力してください。 

3.  **_カードタイプ_**項目で**_企業_**または**_個人_**を選択してください。

    -   **_法人_**：法人カードには、会社名が明記されています。 事業者登録番号と会社名を入力する必要があります。
    -   **_個人_**：個人カードには、カード所有者の名前が入力されています。 会社から支給された場合でも、個人名が明記されていれば個人カードです。 生年月日を8桁で入力してください。

4.  カード情報の入力が完了したら**_請求書の受信者設定_**ボタンをクリックしてください。

5.  請求書を受信する担当者情報を順番に入力し**_完了_**ボタンをクリックしてください。

:::note

複数のメールで請求書を受信する場合は、メールアドレスにコンマ(,)を使用して複数入力できます。 例, a@whatap.io, b@whatap.io, c@whatap.io

:::

#### 海外ユーザー

1.  決済方法を<Cmdname sid="payment_type_card_stripe" className="uitext" />で選択し、<Cmdname sid="next" className="uitext" />ボタンを選択してください。

2.  クレジットカード所有者の名前とクレジットカード番号、有効期限、CVC番号を順番に入力してください。

3.  チェックされた約款情報を確認してください。

4.  カード情報の入力が完了したら**_請求書の受信者設定_**ボタンをクリックしてください。

5.  請求書を受信する担当者情報を順番に入力し**_完了_**ボタンをクリックしてください。

:::note

複数のメールで請求書を受信する場合は、メールアドレスにコンマ(,)を使用して複数入力できます。 例, a@whatap.io, b@whatap.io, c@whatap.io

:::

### 税金計算書

1.  [WhaTapモニタリングサービス](https://service.whatap.io)で**決済**権限を持つアカウントでログインしてください。

2.  ホーム画面の左下にある**_利用履歴_**を選択してください。

    <ImgLang img='tax_invoice_1.png' desc='利用履歴' />

3.  **決済情報\***ページへ移動すると、**_決済情報登録_**ボタンをクリックしてください。

4.  **_決済情報の登録_**ウィンドウが表示されたら、まず国を選択してください。

5.  会社名または法人名を入力してください。

6.  **_電話番号_**と**_請求書の言語_**項目を確認し、必要に応じて変更してください。

7.  所属部署名を入力してください。 これは必須ではありません。

8.  **_次へ_**ボタンを選択してください。

#### 国内ユーザー

1.  決済方法を**_請求書発行_**で選択し、**_次へ_**ボタンを選択してください。

2.  請求書の発行情報を入力してください。

    -   営業許可証に記載されている法人名、代表者名、事業者登録番号、電子請求書発行メールの順に入力してください。

    -   請求書の発行日は次のとおりです。

        -   **_25日_**：利用月の25日に請求書を発行します。
        -   **_月末_**：利用月の月末の日付で請求書を発行します。
        -   <Cmdname sid="date.0" className="uitext" />：利用月を基準として、翌月1日に請求書を発行します。

    -   必要に応じて、業種と業態を入力してください。

3.  請求書の入力が終わりましたら、**_請求書の受信者設定_**ボタンをクリックしてください。

4.  請求書を受信する担当者情報を順番に入力し**_完了_**ボタンをクリックしてください。

    **_CCメール_**CC項目に受信するメールアドレスを追加できます。 2つ以上入力する場合は、コンマ(,)を区切り文字として使用してください。

:::note

-   事業者登録番号を入力し**_認証する_**ボタンをクリックして認証を進めてください。

-   請求書の再発行を希望の場合は、billing@whatap.io までご連絡ください。

:::

#### 海外ユーザー

1.  決済方法**_請求書の発送_**で選択し、**_次へ_**ボタンを選択してください。

2.  **_担当者名_**と**_メール_**項目を確認してください。 必要に応じて変更できます。 

    **_CCメール_**CC項目に受信するメールアドレスを追加できます。 2つ以上入力する場合は、コンマ(,)を区切り文字として使用してください。

3.  請求書の発行日を選択してください。 基準は次のとおりです。

    -   **_25日_**：利用月の25日に請求書を発行します。

    -   **_月末_**：利用月の月末の日付で請求書を発行します。

    -   <Cmdname sid="date.0" className="uitext" />：利用月を基準として、翌月1日に請求書を発行します。

4.  **_住所_**項目を入力してください。

5.  **_適用_**ボタンを選択します。

## 請求書受信メールアドレスの修正

ホーム画面 > <Cmdname sid="my_usage" className="uitext" /> > <Cmdname sid="payment_information" className="uitext" />

<ImgLang img='invoice_mail_1.png' desc='決済アカウントの修正' />

連絡先を変更し、請求先を変更する場合は、請求先アカウントを編集します。 

<Cmdname sid="payment_information" className="uitext" />でメールアドレスの右側にある<LinkImage img="ico-edit.svg" className='ico' /><Cmdname sid="edit" className="uitext" />ボタンをクリックしてください。 <Cmdname sid="billing_info_edit" className="uitext" />ウィンドウが表示されたら、担当者情報(<Cmdname sid="invoice_name" className="uitext" />、<Cmdname sid="email" className="uitext" />、<Cmdname sid="invoice_cc" className="uitext" />)を順番に修正し、<Cmdname sid="registration" className="uitext" />ボタンをクリックしてください。

:::note

複数のメールで請求書を受信する場合は、メールアドレスにコンマ(,)を使用して複数入力できます。 必ずカンマの間にスペースを入れずに入力してください 例、a@whatap.io,b@whatap.io,c@whatap.io

:::

## プロジェクト有償切替{#project-subscription}

決済情報の登録が完了したら、**_プロジェクト有料切替_**メニューに移動します。 ユーザーが登録したプロジェクト全体の一覧を確認できます。

<ImgLang img='billing-pay-list.png' desc='プロジェクト有償切替' />

-   プロジェクトステータスが無料、トライアル、使用制限の場合は**_有料切替_**ボタンがアクティブになります。 プロジェクトを有料に切り替えて使用する場合は、次の操作を行ってください。

    -   各プロジェクト項目の**_有料切替_**ボタンをクリックすると、個別に有料化することができます。 
    -   **_有料_**状態でなく、プロジェクト全体を一括で有料に切替する場合は、![指示線1](/img/number-01.png)**_有料切替_**ボタンをクリックしてください。
    -   各プロジェクト項目の左側にあるチェックボックスを選択し、**_有料切替_**ボタンをクリックしてください。 複数のプロジェクトを選択できます。

    **_有料切替_**ウィンドウが表示されたら、プロジェクトリストを確認して**_確認_**ボタンを選択してください。

-   削除プロジェクトを一覧に含める場合は、![指示線1](/img/number-01.png)**_削除プロジェクトを含む_**トグルボタンを選択してください。

-   **決済権限**のないプロジェクトの一覧を確認するには![指示線1](/img/number-01.png)**_決済権限のないプロジェクトを含む_**トグルボタンを選択してください。

-   プロジェクトの一覧をCSVファイルとしてダウンロードするには![指示線1](/img/number-01.png)**_CSVダウンロード_**ボタンをクリックしてください。

:::note

-   **決済権限**は、有料プランへの切り替え、支払い方法の登録、請求書の受取機能です。
-   プロジェクトが有料サービスに切り替えてから 7 日未満の場合は、有料切替をキャンセルできます。 **_有料キャンセル_**ボタンをクリックしてください。
-   有料プランに切り替えてから 7 日以上経過したプロジェクトを削除できる [プロジェクトの削除] ボタンが用意されています。

:::

## 決済情報の確認

ホーム画面 > **_利用内訳_** > **_決済情報_**>

<ImgLang img='credit_card_8.png' desc='決済情報' />

ユーザーが登録した決済情報を確認できます。 **_修正_**ボタンを選択して、別の決済手段に変更することができます。 決済情報登録の詳細については、[次の文書](#register-pay)を参照してください。

請求書は毎月5営業日以内に発送され、実際の決済は25日に行われます。 25日までに入金を行う必要があります。

## 請求通貨の変更

ユーザーが決済情報を登録する際、<Cmdname sid="TTL06080" className="uitext" />項目で選択した国によって請求金額の請求通貨が決まります。 

<ImgLang img='billing-currency-change.png' desc='請求通貨変更' />

1.  <Cmdname sid="my_usage" className="uitext" />><Cmdname sid="payment_information" className="uitext" />メニューに移動してください。

2.  <Cmdname sid="payment_method" className="uitext" />項目から<Cmdname sid="edit" className="uitext" />ボタンを選択してください。

3.  <Cmdname sid="billing_info_edit" className="uitext" />ウィンドウが表示されると<Cmdname sid="change_payment_method" className="uitext" />、<Cmdname sid="TTL06690" className="uitext" />を順番に選択してください。

4.  <Cmdname sid="TTL06080" className="uitext" />項目から希望する国を選択してください。

5.  <Cmdname sid="next" className="uitext" />ボタンを順番に選択し、決済情報を入力してください。

6.  <Cmdname sid="apply" className="uitext" />ボタンを選択してください。

変更した請求通貨は、<Cmdname sid="project_subscription" className="uitext" />、<Cmdname sid="paymentHistory" className="uitext" />、<Cmdname sid="billingPreview" className="uitext" />メニューからご確認いただけます。

納付履歴、未納金、請求履歴を確認できます。 請求書は、決済情報に入力した請求先のメールアドレスでも確認できます。

<ImgLang img='usage-estimate-history.png' desc='請求及び納付履歴' />

一覧の右端にある**_請求書_**コラムの**_詳細_**ボタンを選択します。

:::note

WhaTapモニタリングサービスは時間単位で課金します。 毎月の基準時間は、対象月の日数✕24時間で計算されます。 以下は、月単位の時間を計算したものです。

-   28日：28✕24 = 672時間
-   29日：29✕24 = 696時間
-   30日：30✕24 = 720時間
-   31日：31✕24 = 744時間

:::

## WhaTapモニタリングサービスの利用料金の計算方法

WhaTapサービスは、後払い決済方法を導入しています。 毎月5営業日以内に前月利用金額の請求書を**決済**権限のあるアカウントのメールで受け取ることができます。 前月の利用期間は、1日から末日までの利用分です。 請求書は、アカウントに登録されている有料サービスのリストに基づいてアカウントごとに作成します。

有料サービスを月の途中から加入した場合、利用した期間を日割り計算して請求します。
