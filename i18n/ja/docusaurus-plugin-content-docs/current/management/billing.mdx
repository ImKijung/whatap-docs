WhaTap監視サービスの無料試用期間を過ぎた場合、またはサービスの無料利用基準を超えた場合は、決済情報を登録し、有料サービスに切り替える必要があります。

## 決済情報の登録 {#register-pay}

お支払い方法は、カードの定期購入またはインボイス対応請求書の2種類からお選びいただけます。 決済を進める前に、次の事項を事前に確認してください。

-   **決済**権限アカウント

-   カード決済の場合、決済カード番号及び有効期限

-   インボイス、事業者登録証、請求書受取メールの発行の場合

:::note

**決済**権限の詳細については、[次の文書](../project/project-structure#project-auth)を参照してください。

:::

### クレジットカード {#credit}

1.  [WhaTapモニタリングサービス](https://service.whatap.io)で**決済**権限を持つアカウントでログインしてください。

2.  ホーム画面の左下にある<Cmdname sid="my_usage" className="uitext" />を選択してください。

    <ImgLang img='tax_invoice_1.png' desc='利用履歴' />

3.  <Cmdname sid="payment_information" className="uitext" />ページへ移動すると、<Cmdname sid="register_billing_info" className="uitext" />ボタンをクリックしてください。

4.  <Cmdname sid="register_billing_info" className="uitext" />ウィンドウが表示されたら、まず国を選択してください。

5.  会社名または法人名を入力してください。

6.  <Cmdname sid="phone" className="uitext" />と<Cmdname sid="TTL06768" className="uitext" />項目を確認し、必要に応じて変更してください。

7.  所属部署名を入力してください。 これは必須ではありません。

8.  <Cmdname sid="next" className="uitext" />ボタンを選択してください。

#### 韓国内ユーザー {#inneruser}

1.  決済方法を<Cmdname sid="credit_card_iamport" className="uitext" />で選択し、<Cmdname sid="next" className="uitext" />ボタンを選択してください。

2.  クレジットカードのカード番号、有効期限、パスワードの最初の2桁の数字を順番に入力してください。 

3.  <Cmdname sid="card_type" className="uitext" />項目で<Cmdname sid="corporation" className="uitext" />または<Cmdname sid="individual" className="uitext" />を選択してください。

    -   <Cmdname sid="corporation" className="uitext" />：法人カードで、カードに社名が入力されています。 組織識別番号と会社名を入力する必要があります。
    -   <Cmdname sid="individual" className="uitext" />：個人カードで、カードにカード所有者の名前が入力されています。 会社から支給された場合でも、個人名が明記されていれば個人カードです。 生年月日を8桁で入力してください。

4.  カード情報の入力が完了したら、<Cmdname sid="bill_recipient_setting" className="uitext" />ボタンをクリックしてください。

5.  請求書を受け取る担当者情報を順番に入力し、<Cmdname sid="complete" className="uitext" />ボタンをクリックしてください。

:::note

複数のメールで請求書を受信する場合は、メールアドレスにコンマ(,)を使用して複数入力できます。 例, a@whatap.io, b@whatap.io, c@whatap.io

:::

#### 韓国外ユーザー {#outteruser}

1.  決済方法を<Cmdname sid="payment_type_card_stripe" className="uitext" />で選択し、<Cmdname sid="next" className="uitext" />ボタンを選択してください。

2.  クレジットカード所有者の名前とクレジットカード番号、有効期限、CVC番号を順番に入力してください。

3.  チェックされた約款情報を確認してください。

4.  カード情報の入力が完了したら、<Cmdname sid="bill_recipient_setting" className="uitext" />ボタンをクリックしてください。

5.  請求書を受け取る担当者情報を順番に入力し、<Cmdname sid="complete" className="uitext" />ボタンをクリックしてください。

:::note

複数のメールで請求書を受信する場合は、メールアドレスにコンマ(,)を使用して複数入力できます。 例, a@whatap.io, b@whatap.io, c@whatap.io

:::

    <ImgLang img='tax_invoice_1.png' desc='利用履歴' />

3.  <Cmdname sid="payment_information" className="uitext" />ページへ移動すると、<Cmdname sid="register_billing_info" className="uitext" />ボタンをクリックしてください。

4.  <Cmdname sid="register_billing_info" className="uitext" />ウィンドウが表示されたら、まず国を選択してください。

5.  会社名または法人名を入力してください。

6.  <Cmdname sid="phone" className="uitext" />と<Cmdname sid="TTL06768" className="uitext" />項目を確認し、必要に応じて変更してください。

7.  所属部署名を入力してください。 これは必須ではありません。

8.  <Cmdname sid="next" className="uitext" />ボタンを選択してください。

#### 韓国内ユーザー {#inneruserinvoice}

1.  決済方法を<Cmdname sid="payment_type_banktransfer" className="uitext" />で選択し、<Cmdname sid="next" className="uitext" />ボタンを選択してください。

2.  請求書の発行情報を入力してください。

    -   営業許可証に記載されている法人名、代表者名、事業者登録番号、電子請求書発行メールの順に入力してください。

    -   請求書の発行日は次のとおりです。

        -   <Cmdname sid="date.25" className="uitext" />：毎月25日に請求書を発行します。
        -   <Cmdname sid="date.-1" className="uitext" />：毎月最終日を基準に請求書を発行します。
        -   <Cmdname sid="date.0" className="uitext" />：利用月を基準として、翌月1日に請求書を発行します。

    -   必要に応じて、業種と業態を入力してください。

3.  請求書の入力が終わりましたら、<Cmdname sid="bill_recipient_setting" className="uitext" />ボタンをクリックしてください。

4.  請求書を受け取る担当者情報を順番に入力し、<Cmdname sid="complete" className="uitext" />ボタンをクリックしてください。

    <Cmdname sid="invoice_cc" className="uitext" />項目に受信するメールアドレスを追加できます。 2つ以上入力する場合は、コンマ(,)を区切り文字として使用してください。

:::note

-   組織識別番号を入力し<Cmdname sid="BTN06501" className="uitext" />ボタンをクリックして認証を進めてください。

-   請求書の再発行を希望の場合は、billing@whatap.io までご連絡ください。

:::

#### 韓国外ユーザー {#outteruserinvoice}

1.  決済方法を<Cmdname sid="payment_type_banktransfer_abroad" className="uitext" />で選択し、<Cmdname sid="next" className="uitext" />ボタンを選択してください。

2.  <Cmdname sid="TTL06687" className="uitext" />と<Cmdname sid="email" className="uitext" />項目を確認してください。 必要に応じて変更できます。

    <Cmdname sid="invoice_cc" className="uitext" />項目に受信するメールアドレスを追加できます。 2つ以上入力する場合は、コンマ(,)を区切り文字として使用してください。

3.  請求書の発行日を選択してください。 基準は次のとおりです。

    -   <Cmdname sid="date.25" className="uitext" />：毎月25日に請求書を発行します。

    -   <Cmdname sid="date.-1" className="uitext" />：毎月最終日を基準に請求書を発行します。

    -   <Cmdname sid="date.0" className="uitext" />：利用月を基準として、翌月1日に請求書を発行します。

4.  <Cmdname sid="company_address" className="uitext" />項目を入力してください。

5.  <Cmdname sid="BTN07117" className="uitext" />ボタンを選択してください。

ホーム画面 > <Cmdname sid="my_usage" className="uitext" /> > <Cmdname sid="payment_information" className="uitext" />

<ImgLang img='invoice_mail_1.png' desc='決済アカウントの修正' />

連絡先を変更し、請求先を変更する場合は、請求先アカウントを編集します。 

<Cmdname sid="payment_information" className="uitext" />でメールアドレスの右側にある<LinkImage img="ico-edit.svg" className='ico' /><Cmdname sid="edit" className="uitext" />ボタンをクリックしてください。 <Cmdname sid="billing_info_edit" className="uitext" />ウィンドウが表示されたら、担当者情報(<Cmdname sid="invoice_name" className="uitext" />、<Cmdname sid="email" className="uitext" />、<Cmdname sid="invoice_cc" className="uitext" />)を順番に修正し、<Cmdname sid="registration" className="uitext" />ボタンをクリックしてください。

## プロジェクト有償切替{#project-subscription}

決済情報の登録が完了したら、<Cmdname sid="project_subscription" className="uitext" />メニューに移動します。 ユーザーが登録したプロジェクト全体の一覧を確認できます。

<ImgLang img='billing-pay-list.png' desc='プロジェクト有償切替' />

-   プロジェクトステータスが無料、トライアル、使用制限の場合は<Cmdname sid="subscribe" className="uitext" />ボタンがアクティブになります。 プロジェクトを有料に切り替えて使用する場合は、次の操作を行ってください。

    -   各プロジェクト項目の<Cmdname sid="subscribe" className="uitext" />ボタンをクリックすると、個別に有料化することができます。 
    -   <Cmdname sid="CNT06012" className="uitext" />状態でなく、プロジェクト全体を一括で有料に切替する場合は、<LinkImage img="number-01.png" desc="지시선 1" /><Cmdname sid="tooltip_limited" className="uitext" />ボタンをクリックしてください。
    -   各プロジェクト項目の左側にあるチェックボックスを選択し、<Cmdname sid="tooltip_limited" className="uitext" />ボタンをクリックしてください。 複数のプロジェクトを選択できます。

    <Cmdname sid="subscribe" className="uitext" />ウィンドウが表示されたら、プロジェクトリストを確認して<Cmdname sid="BTN06053" className="uitext" />ボタンを選択してください。

-   削除プロジェクトを一覧に含めて確認するには![指示線1](/img/number-01.png)<Cmdname sid="include_project_deleted" className="uitext" />トグルボタンを選択してください。

-   **決済権限**のないプロジェクトの一覧を確認するには![指示線1](/img/number-01.png)<Cmdname sid="include_project_has_not_billing" className="uitext" />トグルボタンを選択してください。

-   プロジェクトの一覧をCSVファイルとしてダウンロードするには![指示線1](/img/number-01.png)<Cmdname sid="sms_summary_download_csv" className="uitext" />ボタンをクリックしてください。

:::note

-   **決済権限**は、有料プランへの切り替え、支払い方法の登録、請求書の受取機能です。
-   プロジェクトが有料サービスに切り替えてから 7 日未満の場合は、有料切替をキャンセルできます。 <Cmdname sid="cancel_subscription" className="uitext" />ボタンを選択してください。
-   有料転換時点から7日以降のプロジェクトを削除できる<Cmdname sid="delete_project" className="uitext" />ボタンを提供します。

:::

## WhaTapモニタリングサービスの利用料金の計算方法 {#countbilling}

WhaTapサービスは、後払い決済方法を導入しています。 毎月5営業日以内に前月利用金額の請求書を**決済**権限のあるアカウントのメールで受け取ることができます。 前月の利用期間は、1日から末日までの利用分です。 請求書は、アカウントに登録されている有料サービスのリストに基づいてアカウントごとに作成します。

有料サービスを月の途中から加入した場合、利用した期間を日割り計算して請求します。
