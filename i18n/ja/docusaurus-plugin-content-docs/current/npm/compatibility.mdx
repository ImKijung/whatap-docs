WhaTapの**NPM**を使用するためには、運用中のサーバーに**NPM Agent**をインストールする必要があります。 **NPM**は、OSカーネルが提供するeBPF技術を使用しているため、OSバージョンの確認が必要です。

-   Linuxカーネル5.4バージョン以降に対応します。

-   Linuxリリーズ基準でUbuntu 20.04、CentOS Stream 9以降をお勧めします。

:::note

-   Ubuntu、CentOSの過去のバージョンでもカーネルアップデートを通じて使用できます。

-   その他のリリーズでもカーネルバージョンが合っていれば使用できますが、現在公式にテストされているリリーズはUbuntuとCentOSです。

-   現在Windows、macOSには対応していません。

:::
