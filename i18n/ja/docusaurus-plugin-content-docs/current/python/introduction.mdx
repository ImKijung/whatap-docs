WhaTap Pythonアプリケーションモニタリングは、Pythonベースのウェブアプリケーションサーバーのモニタリングサービスを提供します。Web framework、Server、Batchを同時にモニタリングする韓国唯一の製品です。

主要なウェブフレームワークのアプリケーションに適用でき、DB Query、外部呼び出しを追跡して、どこで呼び出しが遅延しているかを検出することができます。httpまたはhttps以外の特定のトランザクションのデータを追跡する場合、特定のメソッドのモニタリングが可能です。
