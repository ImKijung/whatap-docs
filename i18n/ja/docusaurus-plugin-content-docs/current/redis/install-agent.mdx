## WhaTapデータベースエージェントのダウンロード

{@include: ../common-items/_db-install-agent-steps.mdx} 

3. 解凍したフォルダに移動し、_whatap.conf_ファイルを確認してください。_whatap.conf_ ファイルにプロジェクトのアクセスキー(`license`)情報、WhaTapサーバー情報(`whatap.server.host`)、DB接続情報(`dbms`、`db_ip`、`db_port`)を入力してください。

   ```ini title='whatap.conf'
   license={Access_Key}
   whatap.server.host=13.124.11.223/13.209.172.35 # WhaTap server information

   dbms=redis
   db_ip={DB_server_IP_address}
   db_port={DB_server_port}
   ```
