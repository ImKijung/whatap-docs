2024年07月04日

## Oracle

<Status>New</Status> Oracle V2のリリースに伴う機能を追加

- `ora_hidden_param=true`オプションを設定すると、hidden parameter機能を追加

- `ora_invalid_object=true`オプションを設定すると、invalid objectを収集

- `ora_unusable_index=true`オプションを設定すると、unsable indexを収集

- planの変化チェックと保存機能を追加

- bindパラメータの保存機能を追加

- segment sizeを収集

- `sorts (disk)`、`user rollbacks`メトリクスを追加
