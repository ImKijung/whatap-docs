2023年10月13日

-   <Status>New</Status>プロセス(process)モニタリング機能の追加。プロセスの実行状態がモニタリングできます。

    ```ini title='xos.conf'
    process={process_name}
    ```

-   <Status>New</Status>データベースサーバーのIP一覧の収集。秒単位(最小3秒)で周期(interval)を設定すると収集できます。

    ```ini title='xos.conf'
    iplist_interval={second}
    ```
