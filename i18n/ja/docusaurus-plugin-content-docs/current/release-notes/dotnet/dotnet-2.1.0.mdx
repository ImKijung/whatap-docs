#### 2023年7月6日

-   <Status>New</Status> Web Service設定オプションを追加

    -   **webservice_method_enabled** <span class='type'>Boolean</span>

        デフォルト値`false`

        値を`true`に設定するとWeb Serviceを追跡できます。 設定した値を適用するには、IISを再起動する必要があります。

    -   **webservice_method_prefix** <span class='type'>String</span>

        追跡対象となるメソッド名をネームスペースと一緒に入力してください。 Web Serviceは直接追跡できないため、Web Serviceを使用したメソッドを直接追跡対象として入力する必要があります。 設定した値を適用するには、IISを再起動する必要があります。

        次の例を参照してください。

        ```ini title='whatap.conf'
        webservice_method_prefix=Test.Demo, WebServiceDemo.WebService.
        ```

        :::caution

        -   条件に合ったメソッドがWeb Serviceを使用しなくても追跡対象となります。 ただ、Web Serviceを使用するメソッドでなければ、エラーが発生してもエラーは検出されず、トランザクションが終わっていない状態に維持される可能性があります。

        -   メソッドが所属するクラスまで入力すると、当該クラス内のすべてのメソッドが追跡対象になります。

        :::
