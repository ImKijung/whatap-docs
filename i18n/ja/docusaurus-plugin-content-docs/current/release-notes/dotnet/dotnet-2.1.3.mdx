#### 2023年8月16日

- <Status>Change</Status> Application pool選択方式の変更

  - 既存：モニタリングモジュールがメモリにアップロードされた状態で機能が動作しないようにフィルタリング

  - 変更

    - 対象アプリケーションプールでなければモニタリングモジュールをロードしない

    - モニタリングに関連する何の作業も行わないため、お客様の.NETアプリケーションにいかなる影響も与えない

    - 設定を変更すると、IISを再起動する必要があることに条件を変更

- <Status>Change</Status> サーバ対応環境分析ツールの.NETバージョンを4.61バージョンに下方修正
