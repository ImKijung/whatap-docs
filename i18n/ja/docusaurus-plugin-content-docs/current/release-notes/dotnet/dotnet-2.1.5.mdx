2023年9月13日

- <Status>Fixed</Status>Windows Communication Foundation(以下WCF)とウェブサービスのトランザクション終了時点が断続的に見つからないバグの修正

- <Status>New</Status><code>webservice_method_timeout</code>オプション追加

  WCFとウェブサービス内でExceptionハンドリングがされていない場合、トランザクション終了時点が確認できません。 このような場合に備えて、WCFとウェブサービスは基本的に30秒が過ぎればタイムアウト処理されます。

  <code>webservice_method_timeout</code>オプション値を変更すると、タイムアウト時間を調整できます。 単位はミリセカンド(millisecond, ms)です。 詳細については、<Link to='../../dotnet/agent-webservice'>次の文書</Link>を参照してください。
