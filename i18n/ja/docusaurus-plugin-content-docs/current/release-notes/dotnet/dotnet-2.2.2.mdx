2024年04月18日

- <Status>New</Status> DataAdapter.Fill() メソッドモニタリングを追加

- <Status>New</Status> WebFormモニタリングに対応、ページ移動イベントをトランザクションとして処理

- <Status>Feature</Status>エージェントオプション<code>trace_http_client_ip_header_key</code>追加、詳しい内容は、[次の文書](../../dotnet/agent-number-of-user#trace_http_client_ip_header_key)を参照してください。

- <Status>Fixed</Status> IPアドレスの表示エラーで同時接続者が正しく表示されない現象を修正

- <Status>Deprecated</Status>エージェントがボディを読み込んでいる間に衝突が起こる可能性があるので<code>profile_http_body_enabled</code>オプションの対応を中止
