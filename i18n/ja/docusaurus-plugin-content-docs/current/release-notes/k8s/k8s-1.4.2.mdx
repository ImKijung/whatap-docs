- <Status>Feature</Status><code>kube_pod</code>カテゴリに収集データを追加

  - <code>startTime</code>: Pod開始時刻

  - <code>con_cpu_limit_total</code>: Pod内部コンテナのcpu limitの合計
  - <code>con_mem_alloc_fail_total_count</code>: Pod内部コンテナのmemory割り当て失敗数の合計
