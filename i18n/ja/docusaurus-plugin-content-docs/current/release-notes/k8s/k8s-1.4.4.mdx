- <Status>Changed</Status> client-apiバージョン変更(13.0.2 → 15.0.1)

  低いclient-apiバージョンにより、Kubernetesサービスプラットフォーム(eks、gke)との互換性が取れない現象がありました。 この問題を解決するために、client-apiバージョンをアップデートしました。 特に、deprecatedメソッドをアップデートされたバージョンに置き換えました。

- <Status>Changed</Status><code>kube_entities</code>フィールド名の変更(<code>cronJobs</code> → <code>cronjobs</code>)

- <Status>Fixed</Status>マスターメタ情報CronJobデータ漏れ現象の解決
