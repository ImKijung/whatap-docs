2024年01月16日

- <Status>Feature</Status>オブジェクトマニフェスト収集機能を追加

  1日に1回、ユーザーが設定したタイムゾーンの午前0時にクラスターのオブジェクト情報を収集して保存する機能を追加しました。 この機能を利用するには、次のようにエージェントオプションを設定してください。 詳細については、[次の文書](../../kubernetes/object-manifest)を参照してください。

  - `store_k8s_object_manifest_enabled` <span class='type'>Boolean</span>

    デフォルト値 `true`

    オブジェクトマニフェスト収集機能を活性化することができます。

  - `store_k8s_object_manifest_timezone` <span class='type'>String</span>

    デフォルト値`UTC`

    設定したタイムゾーンの午前0時にオブジェクトマニフェストの収集スケジュールを実行します。

  - `store_k8s_object_manifest_max_fetch_limit` <span class='type'>Int</span>

    デフォルト値`50`

    クラスターで一度に照会するオブジェクト数を設定できます。

- <Status>Fixed</Status>コントロールプレーンモニタリングエージェントの性能を改善。クラスターと通信するクライアントの性能改善を含みます。
