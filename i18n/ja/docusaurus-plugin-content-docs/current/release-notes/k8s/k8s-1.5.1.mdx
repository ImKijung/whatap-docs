-   <Status>Changed</Status> kube workload(ReplicationController)をメトリクス(<code>container</code>、<code>kube_pod</code>、<code>kube_pod_stat</code>), オブジェクトマニフェストに追加

-   <Status>Changed</Status> crioランタイムで<code>overlay_config_path</code>オプションをカスタム設定する場合
    whatap-node-helperコンテナenvに<code>overlay_config_path</code>オプションを追加してパスを設定できるように修正

-   <Status>Fixed</Status>ネームスペースのプロジェクトバージョンの転送間隔を変更：5分間に5秒の転送後timeout → 5分間に5秒の転送し、後は1分ごとに転送
