2024年07月01日

- <Status>Feature</Status>タグカウントを追加

  - **kube_service_pod_mapping**項目を追加
  - **kube_pod**カテゴリに`podUid`タグを追加
  - **kube_service**カテゴリに`externalIp`タグ, PORT(`portName`, `nodePort`, `servicePort`, `targetPort`) フィールドを追加
  - **kube_event**カテゴリに`watch_event_type`、`resource_version`データを追加
  - **kube_event**重複通知現象を削除

- <Status>Feature</Status> **whatap-node-helper**コンテナ環境変数(`env`)に`WHATAP_JAVA_AGENT_PATH`を追加

  :::note

  - 当該環境変数にWhaTap Javaエージェントのjarファイルパスを設定すると、コンテナidと連動できます。 

  - この機能は、[Javaエージェント2.2.33](../java/java-2_2_33)バージョン以降に対応しています。

  :::

:::info

:::
