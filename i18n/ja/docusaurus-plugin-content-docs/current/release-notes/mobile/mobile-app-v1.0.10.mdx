2024年07月04日

- <Status>New</Status> <UI>Home</UI>画面を追加

  ![Mobile app](/img/release/release-mobile-app-add-home.png)

  <UI>Home</UI>メニューでは、お気に入りのプロジェクト、お知らせ、WhaTapブログを確認できます。

- <Status>New</Status>プロジェクトのお気に入り機能を追加

  ![Mobile app](/img/release/release-mobile-app-favorite.png)

  お気に入りを設定したプロジェクトは、<UI>Home</UI>画面とプロジェクト一覧の最上段に表示されます。

- <Status>New</Status>ログイン状態画面を追加

  ![Mobile app](/img/release/release-mobile-app-logining.png)

  ログインの進行状況を詳細に表示し、ネットワークの問題が発生すると、より詳細に把握できます。

- <Status>New</Status>モバイル通知テスト機能を追加

  ![Mobile app](/img/release/release-mobile-app-test-alert.png)
