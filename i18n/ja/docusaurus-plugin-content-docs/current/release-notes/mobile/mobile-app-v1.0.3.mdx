2023年12月19日

WhaTapのモバイルアプリを新たにリリースしました。

- WhaTapモニタリングサービスを通じて見たダッシュボードの主要チャートをモバイルアプリで確認することができます。

- より速くデータを照会して表示できるようになりました。 

- OTP機能を活用してログイン時に二次認証ツールとして活用できます。 

- 組織機能が追加され、組織が持つプロジェクトを確認できます。

- 受信した通知をさらに詳しく確認できます。 

- 英語と日本語に対応しています。
