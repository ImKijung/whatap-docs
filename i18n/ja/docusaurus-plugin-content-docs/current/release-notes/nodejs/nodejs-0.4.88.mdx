2024年01月11日

- <Status>Fixed</Status> <code>trace_http_client_ip_header_key=x-forwarded-for</code>オプションを設定した状態でヘッダで<code>x-forwarded-for</code>値に複数のIPが転送される場合(コンマ基準)、最初の値にIPを設定するよう修正

  :::note

  このオプションの詳細については、[次の文書](../../nodejs/agent-number-of-user#trace_http_client_ip_header_key)を参考にしてください。

  :::

- <Status>Fixed</Status> TCP、UDPソケットモニタリングを修正

- <Status>Fixed</Status> <code>profile_http_header_enabled</code>オプションの値が<code>true</code>であれば、profileのheaders値を表す方式を修正(JSON → key=value\n)

:::info
