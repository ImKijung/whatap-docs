    -   **既存**：コピーしたイベントのタイトルを同じように使用してコピー対象プロジェクトにイベントを重複コピーすると、最後にコピーしたイベントに上書きされます。

    -   **変更**

        -   イベントのタイトルとして`{コピーしたイベント名}(copy)`を使用
        -   コピー対象プロジェクトにイベントを重複コピーすると、コピーした回数だけイベントを作成
