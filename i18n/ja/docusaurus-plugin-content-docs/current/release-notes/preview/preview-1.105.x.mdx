## Preview 1.105.05

2023年7月5日

<div class='indentTab'>

### `共通`

<p><Status>Change</Status>パスワード有効性チェックのロジックを変更</p>

-   最低９桁のアルファベットの小文字および大文字、数字、特殊記号で３種以上の組み合わせ

-   Eメールと同じパスワードの使用不可

-   既存に使用したパスワードと同一のパスワード使用不可

-   連続するキーボード配列に含まれたパスワード使用不可

-   連続するアルファベットを含むパスワード使用不可

-   連続する数字を含むパスワード使用不可

-   現在使用しているパスワードと同一のパスワード使用不可

* * *

### `Container`

<p><Status>Change</Status> <UI>ログ</UI> > <UI>ログ設定</UI>メニューでのノードエージェントログ設定に対し、ユーザーの利便性のためにオプションを直接入力せず、ボタンクリックだけで適用されるように変更</p>

* * *

### `URL`

<p><Status>Fixed</Status> <UI>URL一覧</UI>メニューの<UI>URLを追加する</UI>機能利用時<UI>確認地域</UI>セクションの重複項目を削除</p>

</div>
