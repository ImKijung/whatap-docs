## Preview 1.107.04

2023年8月3日

<div class='indentTab'>

### `共通`

#### ダッシュボード

**_トランザクション分析_**ウィンドウで次の内容を修正します。

-   <Status>Fixed</Status><code>Container</code>プラットフォームのマスタープロジェクトから進入する際に、マルチトランザクション情報が表示されないバグの修正

-   <Status>Fixed</Status><UI>マルチトランザクション</UI>タブ><UI>チャート表示</UI>のプロジェクトフィルター機能の誤作動を修正

-   <Status>Fixed</Status>ランザクション一覧のトランザクションID、マルチトランザクションIDカラム整列誤作動を修正

-   <Status>Fixed</Status><UI>ツリービュー</UI>タブのボタン説明ツールチップ翻訳漏れの修正

    -   **Display Time** → **_시간 표시_**

    -   **Hide Time** → **_시간 숨기기_**

    -   **Display Time Bar** → **_시간바 표시_**

-   <Status>Change</Status>トランザクション一覧の検索機能の対象を変更

    -   変更前:検索対象カラムの指定が必要

    -   変更後:画面に表示されるカラム

* * *

### `Application`

#### アプリケーションダッシュボード

<p><Status>Change</Status><UI>トランザクション分析</UI>機能改編に対するガイドを追加</p>

* * *

### `Database`

#### MySQL V2

<p><Status>Fixed</Status><UI>インスタンスモニタリング</UI>メニューの<UI>Default(rds)</UI>プリセットに含まれているメモリ余分(Cloudwatch)ウィジェットのメトリック変更(<code>FreeStorageSpace</code> → <code>FreeableMemory</code>)</p>

* * *

### `Log`

-   <Status>Feature</Status><UI>アラート通知</UI>><UI>ログ</UI>メニューから<UI>複合ログイベント</UI>でテンプレートイベントを追加する場合、カテゴリが選択されていない状態ではフィルター項目を無効にするように処理
-   <Status>Change</Status><UI>ログ</UI>下位メニューのハイライト機能から選択可能なカラー変更およびログ本文フォントの変更

</div>
