## Preview 1.111.3

2023年10月6日

<div class='indentTab'>

### `共通`

#### User Interface

<p><Status>Fixed</Status>インターバル選択オプションからデフォルト項目のインターバル値を表示する機能を削除</p>

特定のカテゴリでデフォルトのインターバル値が**(0秒)**と表記されますが、実際にチャートに適用されるインターバル値~は0秒ではない場合があり、時間値のテキストを除去

#### お知らせ

<p><Status>Fixed</Status><UI>お知らせ</UI>メニューから掲示物をクリックしてコンテンツ照会時にスクロールが見えないバグを修正</p>

#### ダッシュボード

<p><Status>Changed</Status><UI>トランザクション分析</UI>ウィンドウのレコードタブでmCallerデータに対する案内メッセージを追加</p>

#### プロジェクト一覧

<p><Status>Fixed</Status><UI>プロジェクト一覧</UI>メニューから一覧ビューの時に、CPMプロジェクトが削除されない現象を修正</p>

* * *

### `Application`

<p><Status>Fixed</Status><UI>ダッシュボード</UI>><UI>アプリケーションダッシュボード</UI>メニューの<UI>ヒープメモリ</UI>ウィジェットで詳細表示ウィンドウのタイトルの誤字を修正</p>

* * *

### `Server`

<p><Status>Fixed</Status><UI>サーバー一覧</UI>><UI>サーバー一覧</UI>メニューからエージェント検索時、エージェント一覧の背景色と実際の状態が一致しない問題の修正</p>

* * *

### `Log`

-   <Status>Feature</Status><UI>ログ</UI>><UI>ログ設定</UI>><UI>ログ一次パーサー設定</UI>タブからパーサーを追加する場合、<UI>ログ検出条件</UI>の検索値入力時に、予約語に対する有効性検証を追加

-   <Status>Fixed</Status><UI>ログ</UI>><UI>ログ検索</UI>メニューのログ一覧でタグをクリックすると現れるドロップダウンメニューの<UI>検索</UI>または<UI>除外検索</UI>を選択すると重複値が<UI>フィルター</UI>入力欄に含まれる現像を修正

</div>
