#### Oracle Pro

- <Status>Changed</Status> <Cmdname sid="side_dashboard" className="uitext" /> &gt; <Cmdname sid="side_multiInstance" className="uitext" />メニューで一部のウィジェット名を変更

  - Multidbでなければ中間区切り文字を表示しないように変更

- <Status>Changed</Status> <Cmdname sid="side_event" className="uitext" /> &gt; <Cmdname sid="side_eventSetting" className="uitext" />メニューに進入する時、<Cmdname sid="side_eventSetting" className="uitext" /> <code className="newfunc">New</code>メニューに移動するように変更

- <Status>Fixed</Status> <Cmdname sid="side_dashboard" className="uitext" /> &gt; <Cmdname sid="side_multiInstance" className="uitext" />メニューで***Interconnect Bandwidth***ウィジェットと***Load Balance***ウィジェットのインスタンス順序を、インスタンスセレクタの順序と同一に見えるように修正
