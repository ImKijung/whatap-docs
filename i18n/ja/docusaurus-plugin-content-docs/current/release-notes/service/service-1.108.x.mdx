-   <Status>Feature</Status><UI>アラート通知</UI>><UI>イベント設定</UI>><UI>ログ</UI>><UI>複合ログイベント</UI>機能改善およびログイベントテンプレートの追加

-   <Status>Change</Status><UI>アラート通知</UI>><UI>イベント設定</UI>><UI>ログ</UI>メニューから<UI>リアルタイムログイベント</UI> 追加生成時<UI>検索キー</UI>から<UI>すべて</UI>(all)項目を除去

-   <Status>Fixed</Status><UI>対数検索機</UI>メニューから<code>pcode</code>タグをクリックするとドロップダウンメニューが露出しないように修正

-   <Status>Feature</Status><UI>ログ設定</UI>メニューからカテゴリおよびログ検出条件の選択UIの改善
