## URL Monitoring Agent v2.0.2

リリース：2022-01-05 

### Minor Change

URLモニタリングUser-Agent名を、Java/1.8からWhaTap URL Monitoring Agentに変更しました。 

## URL Monitoring Agent v2.0.1

リリース：2021-10-07

### バグフィックス

-   FullChain証明書を使用していない一部のURLから`sun.security.validator.ValidatorException`が発生する現象を改善しました。

-   旧バージョンでは対応できなかったSHA-256 ECDSAアルゴリズム証明書のURLチェックを有効にするように修正しました。

## URL Monitoring Agent v0.0.3

リリース：2019-04-25

### バグフィックス

httpsプロトコルのURLモニタリングができないため、以下のエラーを改善しました。

```java title='Java'
sun.security.provider.certpath.SunCertPathBuilderException: unable to find valid certification path to requested target
```

SSLベースで通信する場合、証明書を受け取って通常の通信を可能にする仕組みが必要なため、クライアントモジュールが登録したURLにアクセスする際に証明書を許可するよう改善しました。
