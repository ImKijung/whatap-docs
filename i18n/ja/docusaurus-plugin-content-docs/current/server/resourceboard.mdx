ホーム画面 > プロジェクト選択 > **_ダッシュボード_** > **_リソースボード_**

WhaTapモニタリングサービスの初期画面でプロジェクトを選択し **_プロジェクトメニュー_** の下にある **_ダッシュボード_** > **_リソースボード_**を選択してください。

<ImgLang img='server-resourceboard.png' desc='リソースボード'/>

**_リソースボード_**から1つのプロジェクトに登録されているすべてのサーバーをモニタリングできます。 

プロジェクト内のすべてのサーバーの概要とリアルタイムのリソース使用状況の変化を確認できる **_CPU Resource Map_**を提供します。 全体リソース、CPU、メモリ、プロセス使用率の **_Top 5_**を表示します。**_リソースボード_**を使用すると、トラブルを即座に認識して対応することができます。

## ![](/img/number-02.png) 情報パネル

プロジェクトに登録されているすべてのサーバー概要メトリクスを提供します。10秒単位で更新されます。

- **Servers**  

  全体のサーバー数とトラブルのあるサーバーの数を提供します。 領域を選択すると **_サーバーリスト_** ページに移動します。

  - 例外状態：<span class='vslow'>危険</span>、<span class='slow'>警告</span>

  - 正常なサーバー：<span class='snormal'>正常</span>, <span class='k8s-normal'>非アクティブ</span>

- **Total Cores**  

  全体のサーバーのコア(vCPU) の合計です。

- **Avg CPU**  

  全体のサーバーのCPU平均使用率(%)です。 以下のゲージは、平均値、最大値、最小値を示します。

- **Avg Memory**  

  サーバー全体のメモリ平均使用量(%)です。 以下のゲージは、平均値、最大値、最小値を示します。

- **Avg Disk**  

  全体のサーバーのディスクデバイスの平均使用量(%)です。 以下のゲージは、平均値、最大値、最小値を示します。

- **OS**  

  プロジェクト内のlinuxとWindowsオペレーティングシステムの数です。

## ![](/img/number-03.png) CPU Resource Map

全体のサーバーのCPU使用量を示す分布図です。10分間のデータを表示し、10秒ごとに更新されます。 セル色の濃さは、その領域でサーバーのリソースの使用していることを意味します。

- **詳細情報提供**

  セル領域をドラッグすると、詳細情報が表示されます。

  <ImgLang img='server-resourcemap.png' desc='リソースマップ'/>

  - サーバーリストは、範囲内で収集されたデータの最大値でソートされます。

  - 基本の5つのサーバーが選択され、選択されたサーバーの使用状況の傾向をマルチラインチャートで表示します。

  - **_更新_** ボタンを手動でクリックすると、現在のデータに更新できます。 

:::note

**マルチラインチャート**

1つのチャートに複数の線を表示するチャートです。

:::

## ![](/img/number-04.png)リソースイコライザー

**_CPU_**、**_Memory_**、**_Disk I/O_**、**_Disk IOPS_**の上位5つのサーバーの一覧をリアルタイムで表示します。

例えば、Disk I/O(%) メトリクスは、5秒間のディスク使用率を示します。 Disk I/O(%)が80%を超えると、システムの性能に影響を与える可能性があります。警告となるデフォルト値は 90%です。 Disk I/O(%)が100%の場合、ディスクが常時動作していることを意味します。

- イコライザーチャート領域を選択すると、対象**_サーバー詳細_**ページに移動します。 

- ![right-arrow](/img/right-arrow.svg)ボタンを選択すると **_リソースイコライザー_** ページに移動します。 サーバー全体のリアルタイムの使用量が確認できます。
